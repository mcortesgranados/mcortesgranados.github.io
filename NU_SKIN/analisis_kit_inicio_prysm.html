1) Metodologia y limite del comparativo

  • La imagen muestra producto completo + muestras/sobres + accesorios. Por eso no toda pieza visible se puede sumar como si fuera una caja completa.
  • Solo se suman como precio individual los articulos para los que se encontro precio oficial publico y enlace oficial.
  • Cuando el item es visible pero Nu Skin no expone un precio unitario publico localizable, se deja como no publicado y se explica aparte.
  • El precio de miembro por item no siempre esta visible en la pagina publica. Cuando existe en la guia oficial, se reporta; cuando no, se deja en blanco para no inventarlo.
Importante: este documento sirve para analisis comercial. No reemplaza criterio medico, no debe usarse para prometer resultados clinicos y "no consumirlo" no significa dano automatico: significa perder el soporte o la conveniencia especifica que ese item aporta dentro del protocolo.

5) Definicion practica de SV y CV

  • SV: indicador interno de volumen comercial usado por Nu Skin para seguimiento del negocio.
  • CV: valor comisionable interno que Nu Skin usa para calculos del plan comercial.
  • En este kit la imagen reporta SV 500 y CV 525, lo que confirma que el paquete fue pensado mas como herramienta de activacion comercial que como simple compra minorista suelta.

6) Conclusion final

La comparacion correcta no es "suplementos sueltos vs caja bonita". El Kit de Inicio Prysm parece estructurado para vender medicion + narrativa de evidencia + suplementos de soporte. Comercialmente, el kit cobra sentido si el dispositivo se usa para demostrar valor. Si el Prysm iO no se activa en campo, el paquete pierde una parte importante de su logica economica.
