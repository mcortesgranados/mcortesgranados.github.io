Purpose

Automatically determine and place equipment (e.g., grab bars, sinks, toilets, hand dryers) within a room based on:

  • User physical characteristics (height, body type)

  • Accessibility regulations (specifically ADA – Americans with Disabilities Act)

Key Concepts

Aspect Explanation
Auto-placement The system calculates the ideal location for equipment without manual input, positioning it for usability and code compliance.
Height/Body Type Logic Personalizes placement based on target user demographic (e.g., child, adult, wheelchair user), using anthropometric data.
ADA Rules Ensures placements meet U.S. accessibility standards—such as:
• Grab bars 33"–36" AFF (above finished floor)
• Sinks with 27" knee clearance
• Reach range between 15"–48" AFF

Example Use Case

In a bathroom:

  • A wheelchair-accessible sink is placed at 34" height, with space underneath.

  • A hand dryer is mounted at 40", adjusted lower if users are expected to be children.

  • A grab bar is placed according to ADA rules behind and beside a toilet.

Inputs & Outputs

Input Output
Room type and dimensions Valid positions for each equipment item
User body profile (standard adult, wheelchair user, child) Placement height and location suggestions
ADA compliance flag Adjusted positions and types to meet code
