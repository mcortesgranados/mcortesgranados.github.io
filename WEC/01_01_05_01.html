Yes, 1.1.5. 3D rendering setup (lighting presets, batch exports in D5 Render) can be made possible through an API integration. Here's a breakdown of what this operation likely entails:
ObjectiveEnable automated 3D rendering setup, lighting configuration, and batch exporting directly through the Task-Meister CRM system via its API. This allows users to:
  • Set up lighting presets for 3D models.
  • Export multiple renderings (e.g., scenes, designs) in batches using D5 Render (a popular real-time rendering engine).
Key Concepts
Aspect Explanation
Lighting Presets Predefined settings for lighting conditions (e.g., daytime, night, natural light, artificial light) applied to 3D scenes to create realistic visuals.
Batch Exports The ability to export multiple rendered scenes at once, rather than exporting them one by one. This is useful for large projects or design presentations.
D5 Render A real-time 3D rendering engine known for its high-quality visualizations and ease of use. It can handle complex lighting, materials, and environments for architectural designs.
API Integration The system communicates with D5 Render to automate rendering processes such as applying lighting presets and exporting multiple scenes in a batch.
Possible Operations via APIHere are some potential API operations for this task:
API Operation Description Input Output
Set Lighting Preset Apply a predefined or custom lighting preset to a scene. Lighting type (e.g., "sunset", "night", "daytime"), scene ID. Confirmation that preset has been applied to the scene.
Apply Custom Lighting Adjust light sources (intensity, position, color) within a scene manually. Lighting parameters (e.g., intensity, angle, color), scene ID. Updated scene with new lighting settings.
Render Scene Trigger the rendering of a 3D scene with the applied lighting and material settings. Scene ID, resolution settings. Rendered scene file (e.g., image or video output).
Batch Export Export multiple scenes in one batch, with consistent rendering settings. List of scene IDs, format settings (e.g., image format, video format). Batch export confirmation, with rendered files outputted in specified formats.
Set Output Quality Define the quality settings for the rendering process (low, medium, high). Quality setting, scene ID. Confirmation of updated rendering quality settings.
Possible Workflow Example
  1. Lighting Setup:
    You call the Set Lighting Preset API to apply a "daytime" lighting preset to a room scene.
  2. Scene Rendering:
    You trigger the Render Scene API to generate a render of the scene with the applied lighting and material properties.
  3. Batch Export:
    Once multiple rooms or models are ready, you call the Batch Export API to export all the scenes at once, saving time compared to manually exporting each scene.
Inputs & Outputs Example
Input Output
Lighting preset (e.g., "sunset") Applied lighting preset to scene
Scene IDs for batch Confirmation of batch export success, download links to rendered files
API Integration Benefits
  • Efficiency: Automating the lighting setup and batch exports saves time and reduces manual effort in the rendering process.
  • Consistency: Lighting presets and render quality can be applied uniformly across all 3D scenes.
  • Real-time rendering: D5 Render's real-time capabilities enable quick iterations and visual adjustments.
