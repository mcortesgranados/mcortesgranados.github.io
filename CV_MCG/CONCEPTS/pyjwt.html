PyJWT Overview

What is PyJWT?

PyJWT is a Python library for encoding and decoding JSON Web Tokens (JWT). JWT is a compact, URL-safe means of representing claims to be transferred between two parties. It is often used for authentication and information exchange in web development.

Installation:

You can install PyJWT using pip:

pip install PyJWT

Key Features:

  • Encode and decode JWTs using various algorithms (HS256, RS256, etc.).
  • Support for additional JWT features such as expiration time, not-before time, and more.
  • Compatible with Python 2 and 3.
  • Lightweight and easy to use.
  • Provides a flexible and extensible interface for working with JWTs.

Usage Example:

Here's a simple example demonstrating how to encode and decode a JWT using PyJWT:


import jwt
import datetime

# Your secret key
secret_key = "your_secret_key"

# Payload (claims)
payload = {
    "user_id": 123,
    "username": "john_doe",
    "exp": datetime.datetime.utcnow() + datetime.timedelta(days=1)
}

# Encode the JWT
encoded_jwt = jwt.encode(payload, secret_key, algorithm='HS256')
print("Encoded JWT:", encoded_jwt)

# Decode the JWT
try:
    decoded_payload = jwt.decode(encoded_jwt, secret_key, algorithms=['HS256'])
    print("Decoded Payload:", decoded_payload)
except jwt.ExpiredSignatureError:
    print("JWT has expired.")
except jwt.InvalidTokenError:
    print("Invalid JWT.")
