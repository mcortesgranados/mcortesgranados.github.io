Starlette Overview

What is Starlette?

Starlette is a lightweight ASGI (Asynchronous Server Gateway Interface) framework for building web applications with Python. It is designed to be fast, simple, and asynchronous, making it well-suited for building high-performance web services and APIs. Starlette is often used as the foundation for other web frameworks, including FastAPI.

Key Features:

  • ASGI support for handling asynchronous requests and responses.
  • Designed to be modular and extensible, allowing developers to use only the components they need.
  • Includes support for WebSocket handling and HTTP/1.1 and HTTP/1.0 connections.
  • Provides middleware support for adding additional functionality to the application.
  • Compatible with other ASGI servers, such as Uvicorn and Hypercorn.

Installation:

You can install Starlette using pip:

pip install starlette

Usage Example:

Here's a simple example of creating a Starlette application:


from starlette.applications import Starlette
from starlette.responses import JSONResponse
from starlette.routing import Route

async def homepage(request):
    return JSONResponse({"message": "Hello, Starlette!"})

app = Starlette(routes=[
    Route("/", homepage),
])

This example defines a route for the homepage ("/") that returns a JSON response. The Starlette application is then created with the defined route.
