🔍 Objective

Integrate ML/AI models into backend services so that REST APIs can return intelligent, data-driven responses.

✅ Python (FastAPI / Django)

🧠 Scenario

  • Pre-trained ML model using scikit-learn, XGBoost, etc.
  • Saved as .pkl, .joblib, or .h5.
  • Served via FastAPI or Django REST Framework.

🔧 Example with FastAPI

from fastapi import FastAPI
import joblib
import numpy as np

app = FastAPI()
model = joblib.load("models/churn_predictor.pkl")

@app.post("/predict")
async def predict(data: dict):
    features = np.array([data["feature1"], data["feature2"], data["feature3"]])
    prediction = model.predict([features])
    return {"prediction": int(prediction[0])}

✅ Java (Spring Boot)

🧠 Integration Methods

  • Microservice Call: Java app calls a Python ML microservice via HTTP.
  • PMML/ONNX: Load exported model in Java using JPMML or ONNX Runtime.
  • Java ML Libraries: Use libraries like DL4J or Smile natively.

🌐 Example: Calling Python ML API from Spring Boot

public double predictFromPythonService(InputData inputData) {
    String url = "http://localhost:8000/predict";
    HttpEntity<InputData> request = new HttpEntity<>(inputData);
    ResponseEntity<PredictionResponse> response =
        restTemplate.postForEntity(url, request, PredictionResponse.class);
    return response.getBody().getPrediction();
}
