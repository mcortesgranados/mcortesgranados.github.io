🌐 AI in React.js & Angular Applications

1. Product Recommendation (Frontend Integration)

A backend model (FastAPI or Spring Boot) exposes product recommendations via a REST endpoint. In React/Angular, this data is fetched on user login or product browsing. 

// React Example (useEffect)
useEffect(() => {
  fetch(`/api/recommendations?userId=123`)
    .then(res => res.json())
    .then(data => setRecommendations(data));
}, []);

AI was applied on the backend; the frontend simply consumes and renders it in a dynamic component.

2. Predictive Analytics (Dashboard Widgets)

Predictive models for financial trends or user actions were used to show visual forecasts in Angular dashboards. 

// Angular Service to fetch prediction
getForecast(companyId: string): Observable<Forecast[]> {
  return this.http.get<Forecast[]>(`/api/forecast/${companyId}`);
}
// Angular Component
ngOnInit() {
  this.dataService.getForecast('acme-inc').subscribe(data => {
    this.chartData = data;
  });
}

Charts (e.g., Chart.js, Highcharts) rendered future values generated by AI models deployed in Python or Java microservices.

3. AI Model Invocation Flow

How data flows from frontend to AI backend and back:

  1. User interacts with UI (e.g., clicks, selects product)
  2. Frontend sends user context to a REST API
  3. Backend runs ML model or decision engine
  4. Predicted values or recommendations are returned
  5. Frontend dynamically updates view (cards, charts, widgets)

This seamless integration gave the user a “smart” interface while keeping heavy computation on the backend.

4. Example Use Case: Smart Product Dashboard

On login, a React component requests AI-generated suggestions and renders them using conditional logic: 

// React JSX
{recommendations.length > 0 ? (
  <ul>
    {recommendations.map(item => (
      <li key={item.id}>{item.name}</li>
    ))}
  </ul>
) : (
  <p>Loading recommendations...</p>
)}

This is how AI agents invisibly enhanced UX — with real-time, personalized data directly from ML pipelines.

Summary

AI was applied primarily through backend services offering prediction and recommendation APIs. React.js and Angular apps connected to these endpoints, displayed the results intelligently, and updated views dynamically — giving users a rich, personalized experience while maintaining performance and scalability.
