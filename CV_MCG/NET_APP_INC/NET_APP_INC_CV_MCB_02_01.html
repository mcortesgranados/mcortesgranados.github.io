🎨 CSS-in-JS with Styled Components and Emotion

💬 What to Say in the Interview

"In my React projects, I used **CSS-in-JS** libraries like styled-components and @emotion/styled to implement component-level styling. This approach allowed me to write styles directly inside JavaScript, co-located with the components they affected. It made the UI codebase more modular, reusable, and maintainable."

🔧 How I Applied CSS-in-JS

With **CSS-in-JS**, I used libraries to avoid issues of global scope in traditional CSS and gain dynamic and scoped styling for each React component.

Installation

npm install styled-components
# or
npm install @emotion/styled

Example Code

import styled from 'styled-components';

const Button = styled.button\`
  background: \${props => props.primary ? 'blue' : 'gray'};
  color: white;
  padding: 10px 20px;
  border: none;
  border-radius: 8px;
  cursor: pointer;
\`;

function App() {
  return (
    <div>
      <Button primary>Primary Button</Button>
      <Button>Default Button</Button>
    </div>
  );
}

🎨 Theming Support

I used <ThemeProvider> to provide a centralized theme for the entire application, ensuring consistency in colors, fonts, and spacing. This was especially useful in implementing dark/light modes across the app. 

import { ThemeProvider } from 'styled-components';

const theme = {
  primary: '#3498db',
  secondary: '#e74c3c'
};

<ThemeProvider theme={theme}>
  <App />
</ThemeProvider>

🧠 Benefits of Using CSS-in-JS

Feature Benefit
Component-scoped styles No global style leakage
Dynamic styling Easily change styles based on state or props
Theming support Centralized design decisions for consistency
CSS in JavaScript No need for external CSS files
SSR Support Improved performance and SEO with server-side rendering

🔥 Example Use Case

"In a React e-commerce dashboard, I created dynamic product cards using styled-components. Each card changed its background color based on stock level (low, medium, high), and the app automatically adapted to dark/light mode based on the user's preference. This approach streamlined the styling process and made it highly maintainable."
