TypeScript Basic Level Questions and Answers - Part 4

1. How do you use optional chaining in TypeScript?

Optional chaining allows safe access to deeply nested object properties that might be null or undefined.

let user: { name?: { first?: string } } = {};

console.log(user.name?.first); // undefined
user.name = { first: "John" };
console.log(user.name?.first); // John

2. How do you create a tuple in TypeScript?

Tuples allow you to define an array with a fixed number of elements of specific types.

let person: [string, number];
person = ["Alice", 30]; // Valid
// person = [30, "Alice"]; // Error: Type 'number' is not assignable to type 'string'

3. What is a union type in TypeScript?

A union type allows a variable to hold values of different types.

let id: number | string;
id = 123; // Valid
id = "abc"; // Also valid

4. How do you use the Pick utility type in TypeScript?

The Pick utility type allows you to create a new type by picking specific properties from another type.

interface User {
    name: string;
    age: number;
    email: string;
}

type UserNameAndEmail = Pick;

const user: UserNameAndEmail = {
    name: "Alice",
    email: "alice@example.com"
};

5. What is the Exclude utility type in TypeScript?

The Exclude utility type allows you to remove certain types from a union.

type Status = "active" | "inactive" | "pending";
type ActiveStatus = Exclude;

let status: ActiveStatus = "active"; // Only "active" is allowed

6. How do you define index signatures in TypeScript?

Index signatures allow you to define the type of keys and values for dynamic properties in objects.

interface StringArray {
    [index: number]: string;
}

let myArray: StringArray = ["apple", "banana", "cherry"];

console.log(myArray[0]); // "apple"

7. How do you use generics in TypeScript functions?

Generics allow you to create reusable components or functions that work with different types.

function identity(arg: T): T {
    return arg;
}

let output = identity("Hello");
console.log(output); // "Hello"

8. How do you enforce immutability with the ReadonlyArray type in TypeScript?

ReadonlyArray ensures that the elements of an array cannot be modified.

let arr: ReadonlyArray = [1, 2, 3];
// arr.push(4); // Error: Property 'push' does not exist on type 'readonly number[]'

9. How do you use mapped types in TypeScript?

Mapped types allow you to transform the properties of an existing type into another type.

type User = {
    name: string;
    age: number;
};

type ReadonlyUser = {
    [K in keyof User]: Readonly;
};

let user: ReadonlyUser = { name: "Alice", age: 25 };
// user.age = 30; // Error: Cannot assign to 'age' because it is a read-only property

10. How do you define a constructor in a TypeScript class?

The constructor is a special method in a class that gets called when a new instance is created.

class Person {
    name: string;
    age: number;

    constructor(name: string, age: number) {
        this.name = name;
        this.age = age;
    }

    greet() {
        console.log(`Hello, my name is ${this.name}.`);
    }
}

let john = new Person("John", 25);
john.greet(); // "Hello, my name is John."
