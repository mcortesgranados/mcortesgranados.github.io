TypeScript Basic Level Questions and Answers - Part 10

1. How do you create a custom type guard in TypeScript?

A custom type guard allows you to check the type of a variable during runtime.

function isString(value: any): value is string {
    return typeof value === "string";
}

let someValue: any = "hello";
if (isString(someValue)) {
    console.log(someValue.toUpperCase()); // Valid since TypeScript knows it's a string now
}

2. How do you define an abstract class in TypeScript?

An abstract class provides a base class with common functionality, and its methods can be overridden in derived classes.

abstract class Animal {
    abstract makeSound(): void;
    
    move(): void {
        console.log("Moving...");
    }
}

class Dog extends Animal {
    makeSound(): void {
        console.log("Barking...");
    }
}

let myDog = new Dog();
myDog.makeSound(); // Barking...
myDog.move(); // Moving
