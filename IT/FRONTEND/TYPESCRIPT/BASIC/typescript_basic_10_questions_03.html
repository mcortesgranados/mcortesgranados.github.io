TypeScript Basic Level Questions and Answers - Part 3

1. How do you define a class with private members in TypeScript?

Private members in a class are only accessible within the class itself.

class Car {
    private model: string;

    constructor(model: string) {
        this.model = model;
    }

    public getModel(): string {
        return this.model;
    }
}

const myCar = new Car("Toyota");
console.log(myCar.getModel()); // Toyota
// console.log(myCar.model); // Error: Property 'model' is private and only accessible within class 'Car'

2. What is type assertion in TypeScript?

Type assertion allows you to specify a type for a value. It does not perform any type checking or restructuring.

let value: any = "Hello, world!";
let length: number = (value as string).length;

console.log(length); // 13

3. How do you create a function type in TypeScript?

You can define a type for a function using the following syntax.

type MathOperation = (x: number, y: number) => number;

const add: MathOperation = (x, y) => x + y;

console.log(add(5, 3)); // 8

4. What is the unknown type in TypeScript?

The unknown type is a safer alternative to any as it requires type checking before usage.

let input: unknown = "A string";
if (typeof input === "string") {
    console.log(input.toUpperCase()); // A STRING
}

5. How do you use type guards in TypeScript?

Type guards are used to narrow down the type of a variable.

function isString(value: any): value is string {
    return typeof value === "string";
}

let input: unknown = "Hello";

if (isString(input)) {
    console.log(input.toUpperCase()); // HELLO
}

6. What is the keyof operator in TypeScript?

The keyof operator creates a type consisting of the keys of another type.

interface Person {
    name: string;
    age: number;
}

type PersonKeys = keyof Person; // "name" | "age"

7. How do you use the Record utility type in TypeScript?

The Record utility type creates an object type with specific keys and values.

type PageInfo = Record;

const pageInfo: PageInfo = {
    title: "Home Page",
    description: "Welcome to the homepage"
};

8. How do you define a method inside an interface in TypeScript?

Methods in an interface can be defined like any other property, but they include parentheses for arguments.

interface Calculator {
    add(x: number, y: number): number;
}

const myCalculator: Calculator = {
    add: (x, y) => x + y
};

console.log(myCalculator.add(5, 7)); // 12

9. What are conditional types in TypeScript?

Conditional types allow you to create types based on conditions.

type IsString = T extends string ? "Yes" : "No";

type Test1 = IsString; // "Yes"
type Test2 = IsString; // "No"

10. How do you use intersection types in TypeScript?

Intersection types combine multiple types into one.

interface Person {
    name: string;
}

interface Contact {
    email: string;
}

type PersonWithContact = Person & Contact;

const person: PersonWithContact = {
    name: "Alice",
    email: "alice@example.com"
};
