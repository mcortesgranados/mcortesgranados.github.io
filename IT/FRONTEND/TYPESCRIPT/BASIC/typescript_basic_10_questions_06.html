TypeScript Basic Level Questions and Answers - Part 6

1. How do you define an enum in TypeScript?

An enum is a way to define a set of named constants in TypeScript.

enum Direction {
    Up,
    Down,
    Left,
    Right
}

let move: Direction = Direction.Up;
console.log(move); // 0 (enum value)

2. How do you use the instanceof operator in TypeScript?

The instanceof operator checks if an object is an instance of a class.

class Car {
    make: string;
    constructor(make: string) {
        this.make = make;
    }
}

let myCar = new Car("Toyota");
console.log(myCar instanceof Car); // true

3. What is type assertion in TypeScript?

Type assertion is a way to tell TypeScript to treat a value as a specific type, using the as keyword.

let value: any = "This is a string";
let strLength: number = (value as string).length;
console.log(strLength); // 16

4. How do you use a readonly property in TypeScript?

The readonly modifier ensures that a property cannot be modified after initialization.

class Person {
    readonly name: string;

    constructor(name: string) {
        this.name = name;
    }
}

let person = new Person("Alice");
// person.name = "Bob"; // Error: Cannot assign to 'name' because it is a read-only property

5. How do you define a promise in TypeScript?

A Promise in TypeScript is an object that represents a future value, with type safety.

function fetchData(): Promise {
    return new Promise((resolve, reject) => {
        setTimeout(() => {
            resolve("Data loaded");
        }, 1000);
    });
}

fetchData().then(data => console.log(data)); // "Data loaded" (after 1 second)

6. How do you declare a variable with union types in TypeScript?

Union types allow a variable to take on more than one type.

let value: number | string;
value = 42;    // Valid
value = "hello"; // Also valid

7. How do you use the Record utility type in TypeScript?

The Record utility type creates an object type with a set of keys and values of a specific type.

type Roles = "admin" | "user" | "guest";
type UserAccess = Record;

const access: UserAccess = {
    admin: true,
    user: true,
    guest: false
};
console.log(access); // { admin: true, user: true, guest: false }

8. What is an index type in TypeScript?

An index type allows you to use a type as a key in an object, commonly used with keyof.

interface Person {
    name: string;
    age: number;
}

let personKey: keyof Person;
personKey = "name"; // Valid
// personKey = "address"; // Error: Type '"address"' is not assignable to type '"name" | "age"'

9. How do you use the Omit utility type in TypeScript?

The Omit type allows you to create a new type by excluding certain properties from another type.

interface Employee {

    id: number;
    name: string;
    age: number;
    salary: number;
}

type EmployeeWithoutSalary = Omit;

const employee: EmployeeWithoutSalary = {
    id: 1,
    name: "Alice",
    age: 30
};
console.log(employee); // { id: 1, name: "Alice", age: 30 }

10. How do you declare an abstract method in TypeScript?

An abstract method is a method declared in an abstract class, and it must be implemented by derived classes.

abstract class Animal {
    abstract makeSound(): void;
}

class Dog extends Animal {
    makeSound() {
        console.log("Woof!");
    }
}

let dog = new Dog();
dog.makeSound(); // Woof!
