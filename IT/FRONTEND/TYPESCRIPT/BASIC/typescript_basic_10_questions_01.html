TypeScript Basic Level Questions and Answers

1. What is TypeScript?

TypeScript is a superset of JavaScript that adds static types to the language. It compiles down to plain JavaScript and helps in catching type-related errors during development.

2. How do you declare a variable in TypeScript?

let variableName: type = value;

Example:

let age: number = 30;

3. What is an interface in TypeScript?

An interface in TypeScript defines the shape of an object. It can include properties and methods.

interface Person {
    name: string;
    age: number;
}

const person: Person = {
    name: "John",
    age: 25
};

4. How do you define a function in TypeScript?

Functions in TypeScript can include type annotations for parameters and return values.

function greet(name: string): string {
    return `Hello, ${name}`;
}

const message: string = greet("Alice");

5. What is a tuple in TypeScript?

A tuple is an array with fixed sizes and types for each element.

let tuple: [string, number] = ["Hello", 10];

6. How do you handle optional parameters in TypeScript functions?

Optional parameters are denoted with a question mark ? after the parameter name.

function multiply(a: number, b?: number): number {
    return b ? a * b : a;
}

7. What is a class in TypeScript?

A class in TypeScript is a blueprint for creating objects with shared properties and methods.

class Animal {
    name: string;

    constructor(name: string) {
        this.name = name;
    }

    speak(): void {
        console.log(`${this.name} makes a noise.`);
    }
}

const dog = new Animal("Dog");
dog.speak();

8. What is the difference between interface and type in TypeScript?

Both interface and type can be used to define the shape of an object, but type can also be used to define unions, intersections, and more complex types.

interface User {
    name: string;
}

type UserType = {
    name: string;
    age: number;
};

9. How do you create a union type in TypeScript?

A union type allows a variable to be one of several types.

let value: string | number;

value = "Hello"; // valid
value = 100;     // valid
value = true;    // error

10. How do you use generics in TypeScript?

Generics allow you to create reusable and flexible components that work with any data type.

function identity(arg: T): T {
    return arg;
}

let output = identity("myString");
