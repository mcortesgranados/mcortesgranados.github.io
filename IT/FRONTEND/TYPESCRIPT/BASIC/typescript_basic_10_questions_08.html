TypeScript Basic Level Questions and Answers - Part 8

1. How do you create an interface that extends another interface in TypeScript?

You can use the extends keyword to create an interface that inherits properties from another interface.

interface Person {
    name: string;
    age: number;
}

interface Employee extends Person {
    employeeId: number;
}

let employee: Employee = { name: "Alice", age: 30, employeeId: 1234 };
console.log(employee); // { name: "Alice", age: 30, employeeId: 1234 }

2. What is function overloading in TypeScript?

Function overloading allows you to define multiple function signatures for a single function, providing different ways to call it.

function add(a: number, b: number): number;
function add(a: string, b: string): string;
function add(a: any, b: any): any {
    return a + b;
}

console.log(add(1, 2));      // 3
console.log(add("a", "b"));  // ab

3. How do you declare an array of objects in TypeScript?

You can declare an array of objects by specifying the object type in the array declaration.

interface Car {
    make: string;
    model: string;
}

let cars: Car[] = [
    { make: "Toyota", model: "Corolla" },
    { make: "Honda", model: "Civic" }
];

console.log(cars); // [ { make: "Toyota", model: "Corolla" }, { make: "Honda", model: "Civic" } ]

4. How do you create a static method in a TypeScript class?

A static method belongs to the class itself and not to any instance of the class. It can be accessed using the class name.

class MathUtil {
    static square(x: number): number {
        return x * x;
    }
}

console.log(MathUtil.square(4)); // 16

5. What is a namespace in TypeScript, and how do you define one?

A namespace is used to group related code, like functions, interfaces, or classes, to avoid naming collisions.

namespace Utilities {
    export function log(message: string): void {
        console.log(message);
    }

    export function error(message: string): void {
        console.error(message);
    }
}

Utilities.log("Hello, world!"); // Hello, world!

6. How do you enforce the type of an array in TypeScript?

You can declare the type of elements in an array using the square brackets [] or the Array<T> syntax.

let numbers: number[] = [1, 2, 3];
let words: Array = ["hello", "world"];

console.log(numbers); // [1, 2, 3]
console.log(words);   // ["hello", "world"]

7. How do you define a generic function in TypeScript?

Generics allow a function to work with any type while maintaining type safety.

function identity(arg: T): T {
    return arg;
}

console.log(identity(5));         // 5
console.log(identity("hello"));   // hello

8. How do you define an interface for a function in TypeScript?

You can define an interface for a function by specifying its parameter types and return type.

interface Greet {
    (name: string): string;
}

let sayHello: Greet = function(name: string) {
    return `Hello, ${name}`;
};

console.log(sayHello("Alice")); // Hello, Alice

9. How do you use the Partial utility type in TypeScript?

The Partial utility type makes all properties in an interface optional.

interface User {
    id: number;
    name: string;
    email: string;
}

let updateUser: Partial = {
    name: "Bob"
};
console.log(updateUser); // { name: "Bob" }

10. How do you declare a class with multiple constructors in TypeScript?

class Box {
    width: number;
    height: number;

    constructor(width: number, height?: number) {
        this.width = width;
        this.height = height || width; // If height is not provided, set it to width
    }
}

let squareBox = new Box(10);
let rectangleBox = new Box(10, 20);

console.log(squareBox);   // Box { width: 10, height: 10 }
console.log(rectangleBox); // Box { width: 10, height: 20 }
