TypeScript Basic Level Questions and Answers - Part 9

1. How do you use never type in TypeScript?

The never type represents a value that will never occur, often used for functions that never return.

function error(message: string): never {
    throw new Error(message);
}

function fail(): never {
    return error("Something failed");
}
fail();

2. What is a type alias in TypeScript?

A type alias creates a new name for a type. It is especially useful for complex types.

type StringOrNumber = string | number;

let value: StringOrNumber;
value = "Hello";  // Valid
value = 42;       // Valid
// value = true;  // Error: Type 'boolean' is not assignable to type 'string | number'.

3. How do you define a function with a default parameter in TypeScript?

You can provide a default value for a function parameter by assigning it during the function declaration.

function greet(name: string = "Guest"): string {
    return `Hello, ${name}!`;
}

console.log(greet()); // Hello, Guest!
console.log(greet("Alice")); // Hello, Alice!

4. What is a union type in TypeScript?

A union type allows a variable to hold values of multiple types.

let value: number | string;
value = 10; // Valid
value = "Hello"; // Valid
// value = true; // Error: Type 'boolean' is not assignable to type 'number | string'

5. How do you use type assertions in TypeScript?

Type assertions allow you to tell the compiler to treat an entity as a specific type.

let someValue: any = "this is a string";
let strLength: number = (someValue as string).length;

console.log(strLength); // 16

6. How do you define a readonly array in TypeScript?

The readonly keyword ensures that an array's elements cannot be modified after assignment.

let arr: ReadonlyArray = [1, 2, 3];
// arr[0] = 10; // Error: Index signature in type 'readonly number[]' only permits reading
console.log(arr); // [1, 2, 3]

7. How do you declare a tuple with different types in TypeScript?

A tuple is an array with a fixed number of elements where each element has a specific type.

let tuple: [string, number];
tuple = ["Hello", 42]; // Valid
// tuple = [42, "Hello"]; // Error: Type 'number' is not assignable to type 'string'

8. How do you declare an enum in TypeScript?

An enum is a way of defining a set of named constants, either numeric or string-based.

enum Direction {
    Up,
    Down,
    Left,
    Right
}

let dir: Direction = Direction.Up;
console.log(dir); // 0 (index of Up in the enum)

9. How do you implement a function that returns a promise in TypeScript?

You can specify the return type as Promise in TypeScript for functions returning promises.

function fetchData(url: string): Promise<string> {
    return new Promise((resolve, reject) => {
        setTimeout(() => resolve("Data from " + url), 1000);
    });
}

fetchData("https://example.com").then(data => console.log(data));

10. How do you define a conditional type in TypeScript?

Conditional types enable you to create types based on conditions.

type MessageType<T> = T extends string ? "TextMessage" : "DataMessage";

let textMessage: MessageType<string>;  // "TextMessage"
let dataMessage: MessageType<number>;  // "DataMessage"
