TypeScript Intermediate Level Questions and Answers - Part 16

1. How do you use keyof with typeof to create a type based on the keys of an object?

Using keyof with typeof allows you to create a type that represents the keys of an object.

const person = {
    name: "Alice",
    age: 30
};

type PersonKeys = keyof typeof person; // "name" | "age"

const key: PersonKeys = "name";

console.log(key); // name

2. How do you use type to create a union of multiple types?

A union type represents a value that can be one of several types.

type StringOrNumber = string | number;

const value1: StringOrNumber = "Hello";
const value2: StringOrNumber = 42;

console.log(value1); // Hello
console.log(value2); // 42

3. How do you use type to define an object type with optional properties?

Optional properties in a type alias can be defined using the ? modifier.

type Person = {
    name: string;
    age?: number;
};

const person1: Person = { name: "Alice" };
const person2: Person = { name: "Bob", age: 30 };

console.log(person1); // { name: 'Alice' }
console.log(person2); // { name: 'Bob', age: 30 }

4. How do you use type to define a function type that can return different types based on input?

Conditional types can be used to define function types that return different types based on input.

type Response = T extends boolean ? "Success" : "Failure";

const result1: Response = "Success";
const result2: Response = "Failure";

console.log(result1); // Success
console.log(result2); // Failure

5. How do you use interface to extend another interface?

Interfaces can extend other interfaces to inherit properties and methods.

interface Person {
    name: string;
    age: number;
}

interface Contact extends Person {
    email: string;
}

const contact: Contact = {
    name: "Alice",
    age: 30,
    email: "alice@example.com"
};

console.log(contact); // { name: 'Alice', age: 30, email: 'alice@example.com' }

6. How do you use type to create a type that represents an array of a specific type?

Type aliases can define an array with a specific element type.

type NumberArray = number[];

const numbers: NumberArray = [1, 2, 3, 4];

console.log(numbers); // [1, 2, 3, 4]

7. How do you use type to define a type that can be either an object or a function?

Union types allow you to define a type that can be multiple different types.

type ObjectOrFunction = { name: string } | (() => void);

const obj: ObjectOrFunction = { name: "Alice" };
const func: ObjectOrFunction = () => console.log("Hello");

console.log(obj); // { name: 'Alice' }
func(); // Hello

8. How do you use type to define a type with a method that returns a promise?

You can define a type for a method that returns a promise using type aliases.

type AsyncMethod = () => Promise;

const asyncMethod: AsyncMethod = async () => "Hello, TypeScript!";

asyncMethod().then(result => console.log(result)); // Hello, TypeScript!

9. How do you use type to create a type for an object with dynamic keys and specific value types?

Type aliases can define objects with dynamic keys and specified value types.

type NumericDictionary = {
    [key: string]: number;
};

const dictionary: NumericDictionary = {
    a: 1,
    b: 2
};

console.log(dictionary); // { a: 1, b: 2 }

10. How do you use type to define a type with a method that accepts a generic type?

Type aliases can define methods with generic types for flexibility.

type GenericFunction = (value: T) => T;

const stringFunction: GenericFunction = (value) => value.toUpperCase();
const numberFunction: GenericFunction = (value) => value + 1;

console.log(stringFunction("hello")); // HELLO
console.log(numberFunction(10)); // 11

11. How do you use type to define a type that represents a tuple with mixed types?

Tuples allow you to define an array with a fixed number of elements, each of a specific type.

type MixedTuple = [string, number, boolean];

const tuple: MixedTuple = ["Alice", 30, true];

console.log(tuple); // ["Alice", 30, true]

12. How do you use type to define a type that represents an object with optional methods?

Type aliases can include optional methods by marking them with ?.

type Person = {
    name: string;
    greet?: () => void;
};

const person1: Person = {
    name: "Alice"
};

const person2: Person = {
    name: "Bob",
    greet: () => console.log("Hello!")
};

console.log(person1); // { name: 'Alice' }
person2.greet(); // Hello!

13. How do you use type to define a type that represents a function with specific parameter types?

Function types in type aliases can specify parameter types explicitly.

type AddFunction = (a: number, b: number) => number;

const add: AddFunction = (a, b) => a + b;

console.log(add(5, 3)); // 8

14. How do you use type to define a type that represents an object with properties of different types?

Type aliases can define objects with properties of various types.

type Profile = {
    name: string;
    age: number;
    active: boolean;
};

const profile: Profile = {
    name: "Alice",
    age: 30,
    active: true
};

console.log(profile); // { name: 'Alice', age: 30, active: true }

15. How do you use type to define a type that represents a function with variable argument types?

Type aliases can define functions with variable argument types using rest parameters.

type ConcatFunction = (...args: string[]) => string;

const concat: ConcatFunction = (...args) => args.join(" ");

console.log(concat("Hello", "TypeScript")); // Hello TypeScript
