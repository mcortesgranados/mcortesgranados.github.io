Azure Services (PaaS, IaaS, iPaaS) Q&A - Set 9

1. What is Azure SQL Database?

Azure SQL Database is a fully managed relational database service based on the latest stable version of the Microsoft SQL Server database engine, optimized for cloud applications.

2. What is Azure Cognitive Services?

Azure Cognitive Services are a set of AI services and APIs that help developers embed AI capabilities such as vision, speech, language, and decision-making into their applications.

3. What is Azure Table Storage?

Azure Table Storage is a service that stores structured NoSQL data in the cloud, providing a key/attribute store with a schemaless design for rapid development and flexible scaling.

4. What is Azure Logic Apps vs. Azure Functions?

Azure Logic Apps is designed for workflow automation with connectors, while Azure Functions is serverless computing focused on executing event-driven code with custom logic.

5. What is Azure Cost Management and Billing?

Azure Cost Management and Billing is a suite of tools that allows you to track, manage, and optimize your cloud spend, offering insights into cost and usage patterns.

6. What is Azure Traffic Manager?

Azure Traffic Manager is a DNS-based traffic load balancer that distributes traffic optimally across services worldwide, improving availability and responsiveness.

7. What is Azure Batch?

Azure Batch is a high-performance computing (HPC) service that allows you to run large-scale parallel and batch computing jobs in the cloud, managing and scaling your compute nodes.

8. What is Azure Active Directory (Azure AD)?

Azure Active Directory is a cloud-based identity and access management service that provides authentication and authorization for users and applications in Azure and other services.

9. What is Azure Data Explorer?

Azure Data Explorer is a fast and highly scalable data analytics service that helps you explore and analyze large volumes of diverse data from IoT devices, apps, and more.

10. What is Azure Arc?

Azure Arc enables you to manage and secure your on-premises, multi-cloud, and edge environments from a single control plane, extending Azure management and services anywhere.

11. What is Azure Purview?

Azure Purview is a unified data governance service that helps you discover, manage, and govern your on-premises, multi-cloud, and software-as-a-service (SaaS) data.

12. What is Azure DevOps?

Azure DevOps is a set of development tools that support the entire DevOps lifecycle, including CI/CD pipelines, version control, agile planning, and more, integrated into a single platform.

13. What is Azure Cognitive Search?

Azure Cognitive Search is a fully managed search service with built-in AI capabilities, enabling powerful full-text search, indexing, and data enrichment of structured and unstructured data.

14. What is Azure Marketplace?

Azure Marketplace is an online store that offers software applications, developer services, and consulting services that are certified and optimized to run on Azure.

15. What is Azure Bastion?

Azure Bastion provides secure and seamless RDP and SSH connectivity to your virtual machines directly through the Azure portal, without exposing your VM to public IP addresses.

16. What is Azure Lab Services?

Azure Lab Services allows you to set up and provision Windows or Linux virtual labs for your classes, training, and other scenarios, managing lab environments with ease.

17. What is Azure Virtual Desktop?

Azure Virtual Desktop is a comprehensive desktop and app virtualization service that runs in the cloud, allowing secure remote access to desktop environments from anywhere.

18. What is Azure Blueprints?

Azure Blueprints allows you to define a repeatable set of Azure resources that implement and adhere to organizational standards, patterns, and compliance requirements.

19. What is Azure Communication Services?

Azure Communication Services enables developers to add voice, video, chat, and SMS capabilities to their applications, leveraging the same reliable, secure platform used by Microsoft Teams.

20. What is Azure Managed Disks?

Azure Managed Disks are block-level storage volumes managed by Azure that are used with Azure VMs, providing high availability, scalability, and security for your data.
