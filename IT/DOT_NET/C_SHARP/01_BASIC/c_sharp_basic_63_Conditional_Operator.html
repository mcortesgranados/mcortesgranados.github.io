C# Conditional Operator

The conditional operator in C# is a ternary operator that provides a shorthand way to perform conditional logic. It is often used to simplify expressions where a value is chosen based on a condition.

1. What is the Conditional Operator?

The conditional operator, represented by ?:, is a ternary operator that evaluates a condition and returns one of two values depending on whether the condition is true or false.

2. Syntax

The syntax for the conditional operator is as follows:


condition ? value_if_true : value_if_false;

Here, condition is an expression that evaluates to true or false. If the condition is true, value_if_true is returned. If the condition is false, value_if_false is returned.

3. Example

Hereâ€™s an example demonstrating the use of the conditional operator:


int age = 20;
string result = age >= 18 ? "Adult" : "Minor";

Console.WriteLine(result); // Output: Adult

In this example, the condition age >= 18 is true, so the string "Adult" is assigned to the result variable.

4. Usage in Expressions

The conditional operator can also be used within larger expressions to simplify code:


int number = 10;
int absoluteValue = number >= 0 ? number : -number;

Console.WriteLine(absoluteValue); // Output: 10

Here, the conditional operator is used to determine the absolute value of number. If number is negative, its negation is used; otherwise, the number itself is used.

5. Key Points

  • Definition: The conditional operator ?: allows you to choose between two values based on a condition.
  • Syntax: condition ? value_if_true : value_if_false.
  • Usage: Simplifies conditional logic and can be used within larger expressions for concise code.
