C# Default Parameters

In C#, you can provide default values for parameters in methods. Default parameters allow you to call a method without specifying all arguments, as the method will use the provided default values for missing arguments.

1. Basics of Default Parameters

To use default parameters, specify the default value in the method signature. If the caller does not provide a value for a parameter with a default value, the default value is used.

2. Example of Default Parameters

Here's an example demonstrating the use of default parameters to create a greeting message:


public class Greeting
{
    // Method with default parameters
    public void Greet(string name, string salutation = "Hello", string punctuation = "!")
    {
        Console.WriteLine($"{salutation} {name}{punctuation}");
    }
}

public class Program
{
    public static void Main()
    {
        Greeting greet = new Greeting();

        // Calling with all arguments
        greet.Greet("Alice", "Hi", "?");

        // Calling with default salutation and punctuation
        greet.Greet("Bob");

        // Calling with default punctuation
        greet.Greet("Charlie", "Greetings");
    }
}

Output:


Hi Alice?
Hello Bob!
Greetings Charlie

3. Key Points

  • Default Values: Parameters can be assigned default values in the method signature.
  • Optional Arguments: If an argument is not provided, the default value is used.
  • Order: Default parameters must be specified from right to left in the method signature.

4. Conclusion

Default parameters in C# provide flexibility in method calls by allowing you to omit certain arguments. This feature can simplify method calls and make your code more readable by reducing the number of overloads needed.
