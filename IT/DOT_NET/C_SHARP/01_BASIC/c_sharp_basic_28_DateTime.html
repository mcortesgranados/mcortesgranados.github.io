C# DateTime

The DateTime structure in C# represents date and time. It provides various methods and properties to work with dates and times.

1. Getting the Current Date and Time

You can get the current date and time using the DateTime.Now property:


    using System;

    class Program
    {
        static void Main()
        {
            // Get the current date and time
            DateTime now = DateTime.Now;

            Console.WriteLine("Current Date and Time: " + now);
        }
    }

2. Formatting Dates

To format dates, you can use the ToString() method with format specifiers:


    using System;

    class Program
    {
        static void Main()
        {
            DateTime now = DateTime.Now;

            // Format date in various ways
            Console.WriteLine("Short Date: " + now.ToString("d"));
            Console.WriteLine("Long Date: " + now.ToString("D"));
            Console.WriteLine("Short Time: " + now.ToString("t"));
            Console.WriteLine("Long Time: " + now.ToString("T"));
            Console.WriteLine("Custom Format: " + now.ToString("yyyy-MM-dd HH:mm:ss"));
        }
    }

3. Date Arithmetic

You can perform arithmetic operations on dates, such as adding or subtracting time:


    using System;

    class Program
    {
        static void Main()
        {
            DateTime now = DateTime.Now;

            // Add 10 days
            DateTime futureDate = now.AddDays(10);
            Console.WriteLine("Date in 10 days: " + futureDate);

            // Subtract 5 hours
            DateTime pastDate = now.AddHours(-5);
            Console.WriteLine("Date 5 hours ago: " + pastDate);
        }
    }

4. Parsing Dates

To convert a string to a DateTime object, use the DateTime.Parse() or DateTime.TryParse() methods:


    using System;

    class Program
    {
        static void Main()
        {
            string dateString = "2024-08-28";

            // Parse the date
            DateTime parsedDate = DateTime.Parse(dateString);
            Console.WriteLine("Parsed Date: " + parsedDate);

            // Try parsing with validation
            if (DateTime.TryParse(dateString, out DateTime validDate))
            {
                Console.WriteLine("Valid Date: " + validDate);
            }
            else
            {
                Console.WriteLine("Invalid Date");
            }
        }
    }

5. Key Points

  • DateTime.Now provides the current date and time.
  • ToString() allows formatting dates and times using format specifiers.
  • AddDays(), AddHours(), and similar methods can be used for date arithmetic.
  • DateTime.Parse() and DateTime.TryParse() are used to convert strings to DateTime objects.
