C# Enums

Enums (short for enumerations) in C# are a special "class" that represents a group of named constants. They are useful for defining a set of related constants that can be used in a type-safe manner.

1. Purpose

Enums are used to:

  • Group Related Constants: Define a set of related values that represent a specific category or type.
  • Improve Code Readability: Replace numeric or string constants with meaningful names.
  • Ensure Type Safety: Provide compile-time checking of values.

2. Defining an Enum

To define an enum, use the enum keyword. Here’s an example of a simple enum definition:


using System;

public enum DaysOfWeek
{
    Sunday,
    Monday,
    Tuesday,
    Wednesday,
    Thursday,
    Friday,
    Saturday
}

3. Using Enums

Enums can be used to make code more readable and maintainable. Here’s an example of how to use an enum:


using System;

class Program
{
    static void Main()
    {
        // Assign an enum value
        DaysOfWeek today = DaysOfWeek.Monday;

        // Display the enum value
        Console.WriteLine($"Today is {today}");

        // Use enum in a switch statement
        switch (today)
        {
            case DaysOfWeek.Monday:
                Console.WriteLine("Start of the work week.");
                break;
            case DaysOfWeek.Friday:
                Console.WriteLine("Almost the weekend!");
                break;
            case DaysOfWeek.Saturday:
            case DaysOfWeek.Sunday:
                Console.WriteLine("Weekend!");
                break;
            default:
                Console.WriteLine("Midweek days.");
                break;
        }
    }
}

4. Key Points

  • Underlying Type: By default, enums are based on the int type, but you can specify another integral type.
  • Named Constants: Each value in an enum is a named constant with an associated integer value starting from 0.
  • Explicit Values: You can assign specific values to enum members if needed.
  • Type Safety: Enums ensure that only valid values are used, avoiding errors from invalid literals.
  • Conversion: Enums can be converted to and from their underlying type using casting.
