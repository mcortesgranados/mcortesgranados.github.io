C# Constructors

Constructors in C# are special methods used to initialize objects of a class. They are called automatically when an object is created and allow you to set initial values for the object’s fields.

1. Defining a Constructor

A constructor has the same name as the class and does not have a return type. It can be parameterless or can take parameters to initialize the fields with specific values.


    public class Person
    {
        private string name;
        private int age;

        // Parameterless constructor
        public Person()
        {
            name = "Unknown";
            age = 0;
        }

        // Parameterized constructor
        public Person(string name, int age)
        {
            this.name = name;
            this.age = age;
        }

        public void DisplayInfo()
        {
            Console.WriteLine($"Name: {name}, Age: {age}");
        }
    }

2. Overloading Constructors

Constructor overloading allows a class to have multiple constructors with different parameters. This enables you to create objects in different ways based on the arguments provided.


    public class Person
    {
        private string name;
        private int age;

        // Parameterless constructor
        public Person()
        {
            name = "Unknown";
            age = 0;
        }

        // Constructor with one parameter
        public Person(string name)
        {
            this.name = name;
            age = 0;
        }

        // Constructor with two parameters
        public Person(string name, int age)
        {
            this.name = name;
            this.age = age;
        }

        public void DisplayInfo()
        {
            Console.WriteLine($"Name: {name}, Age: {age}");
        }
    }

3. Using Constructors

When creating an object, you can choose which constructor to use based on the parameters you provide. The constructor initializes the object’s fields with the provided values or default values.


    class Program
    {
        static void Main()
        {
            // Using the parameterless constructor
            Person person1 = new Person();
            person1.DisplayInfo();

            // Using the constructor with one parameter
            Person person2 = new Person("Alice");
            person2.DisplayInfo();

            // Using the constructor with two parameters
            Person person3 = new Person("Bob", 30);
            person3.DisplayInfo();
        }
    }

4. Example Program

Here’s a complete C# program demonstrating the use of constructors, including overloading and initialization:


    using System;

    public class Person
    {
        private string name;
        private int age;

        // Parameterless constructor
        public Person()
        {
            name = "Unknown";
            age = 0;
        }

        // Constructor with one parameter
        public Person(string name)
        {
            this.name = name;
            age = 0;
        }

        // Constructor with two parameters
        public Person(string name, int age)
        {
            this.name = name;
            this.age = age;
        }

        public void DisplayInfo()
        {
            Console.WriteLine($"Name: {name}, Age: {age}");
        }
    }

    class Program
    {
        static void Main()
        {
            // Creating Person objects using different constructors
            Person person1 = new Person();
            Person person2 = new Person("Alice");
            Person person3 = new Person("Bob", 30);
            
            // Displaying information
            person1.DisplayInfo();
            person2.DisplayInfo();
            person3.DisplayInfo();
        }
    }

Explanation

In the example program:

  • The Person class includes parameterless and parameterized constructors, demonstrating constructor overloading.
  • The DisplayInfo method displays the initialized values for each object.
  • The Main method creates instances of the Person class using different constructors and displays their information.
