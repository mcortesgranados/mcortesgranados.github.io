C# Regular Expressions

Regular expressions (regex) are used for pattern matching and manipulation of text. In C#, the System.Text.RegularExpressions namespace provides classes for working with regular expressions.

1. Basic Concepts

Regular expressions use special characters to define patterns. Some common special characters include:

  • . - Matches any single character except newline.
  • * - Matches zero or more occurrences of the previous character.
  • + - Matches one or more occurrences of the previous character.
  • ? - Matches zero or one occurrence of the previous character.
  • [abc] - Matches any one of the characters inside the brackets.
  • [^abc] - Matches any character not inside the brackets.
  • \d - Matches any digit (0-9).
  • \D - Matches any non-digit character.
  • \w - Matches any word character (alphanumeric + underscore).
  • \W - Matches any non-word character.
  • \s - Matches any whitespace character (spaces, tabs, line breaks).
  • \S - Matches any non-whitespace character.

2. Using Regular Expressions in C#

The Regex class in the System.Text.RegularExpressions namespace is used to work with regular expressions. Here are some common methods:

  • Regex.IsMatch() - Determines if a string matches a regular expression pattern.
  • Regex.Match() - Searches for the first occurrence of a pattern in a string.
  • Regex.Matches() - Searches for all occurrences of a pattern in a string.
  • Regex.Replace() - Replaces occurrences of a pattern with a specified replacement string.

3. Examples

Example 1: Checking if a String Matches a Pattern


    using System;
    using System.Text.RegularExpressions;

    class Program
    {
        static void Main()
        {
            string pattern = @"^\d{3}-\d{2}-\d{4}$"; // Pattern for a social security number
            string input = "123-45-6789";

            bool isMatch = Regex.IsMatch(input, pattern);
            Console.WriteLine("Is Match: " + isMatch);
        }
    }

Example 2: Finding Matches in a String


    using System;
    using System.Text.RegularExpressions;

    class Program
    {
        static void Main()
        {
            string pattern = @"\d+"; // Pattern to find all numbers
            string input = "There are 3 apples and 4 oranges.";

            MatchCollection matches = Regex.Matches(input, pattern);
            foreach (Match match in matches)
            {
                Console.WriteLine("Found: " + match.Value);
            }
        }
    }

Example 3: Replacing Text


    using System;
    using System.Text.RegularExpressions;

    class Program
    {
        static void Main()
        {
            string pattern = @"\d+"; // Pattern to find all numbers
            string input = "There are 3 apples and 4 oranges.";
            string replacement = "#";

            string result = Regex.Replace(input, pattern, replacement);
            Console.WriteLine("Result: " + result);
        }
    }

4. Key Points

  • Regular expressions are patterns used to match and manipulate text.
  • Regex class in System.Text.RegularExpressions provides methods for working with regex in C#.
  • Common methods include IsMatch(), Match(), Matches(), and Replace().
