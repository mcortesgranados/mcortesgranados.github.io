C# Event Handlers

Events in C# are a way to provide notifications when certain actions occur, such as a button click or a data update. Event handlers are methods that respond to these events. They are essential for creating interactive applications.

1. Purpose

Event handlers in C# are used to:

  • Respond to User Actions: Handle events like button clicks, text changes, and more.
  • Implement Event-Driven Programming: Design applications where actions trigger responses.
  • Decouple Components: Allow components to react to changes without tight coupling.

2. Defining Events

To define an event, declare a delegate and an event field. Here’s an example:


using System;

public class Publisher
{
    // Define a delegate
    public delegate void NotifyEventHandler(object sender, EventArgs e);

    // Declare an event of the delegate type
    public event NotifyEventHandler Notify;

    // Method to trigger the event
    public void RaiseEvent()
    {
        Notify?.Invoke(this, EventArgs.Empty);
    }
}

3. Creating Event Handlers

An event handler is a method that matches the delegate signature of the event. Here’s how to create and attach an event handler:


using System;

public class Subscriber
{
    public void OnNotify(object sender, EventArgs e)
    {
        Console.WriteLine("Event received!");
    }
}

4. Attaching Event Handlers

To respond to an event, attach the event handler to the event. Here’s an example of how to do this:


using System;

class Program
{
    static void Main()
    {
        Publisher publisher = new Publisher();
        Subscriber subscriber = new Subscriber();

        // Attach the event handler
        publisher.Notify += subscriber.OnNotify;

        // Trigger the event
        publisher.RaiseEvent();
    }
}

5. Key Points

  • Delegates: Events are based on delegates, which define the method signature for event handlers.
  • Event Declaration: Use the event keyword to declare an event in a class.
  • Event Handlers: Methods that match the delegate signature handle the event.
  • Attaching Handlers: Use the += operator to attach event handlers to events.
  • Triggering Events: Call the event to notify all attached handlers.
