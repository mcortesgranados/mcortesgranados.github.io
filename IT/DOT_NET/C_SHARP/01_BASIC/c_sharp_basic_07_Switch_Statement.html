C# Switch Statement

The `switch` statement in C# is used to execute one block of code among many possible blocks based on the value of an expression. It provides a cleaner and more readable way to handle multiple conditions compared to a series of `if-else` statements.

Syntax

The basic syntax of the `switch` statement is:


    switch (expression)
    {
        case value1:
            // Code to execute when expression equals value1
            break;
        case value2:
            // Code to execute when expression equals value2
            break;
        ...
        default:
            // Code to execute when no case matches
            break;
    }

Example 1: Basic Switch Statement

Here is a simple example demonstrating how to use the `switch` statement to determine the day of the week based on an integer value:


    int day = 3;

    switch (day)
    {
        case 1:
            Console.WriteLine("Monday");
            break;
        case 2:
            Console.WriteLine("Tuesday");
            break;
        case 3:
            Console.WriteLine("Wednesday");
            break;
        case 4:
            Console.WriteLine("Thursday");
            break;
        case 5:
            Console.WriteLine("Friday");
            break;
        case 6:
            Console.WriteLine("Saturday");
            break;
        case 7:
            Console.WriteLine("Sunday");
            break;
        default:
            Console.WriteLine("Invalid day");
            break;
    }

Example 2: Switch with Multiple Cases

In this example, multiple cases can share the same block of code:


    char grade = 'B';

    switch (grade)
    {
        case 'A':
        case 'B':
        case 'C':
            Console.WriteLine("You passed.");
            break;
        case 'D':
            Console.WriteLine("You barely passed.");
            break;
        case 'F':
            Console.WriteLine("You failed.");
            break;
        default:
            Console.WriteLine("Invalid grade.");
            break;
    }

Example 3: Switch with Expression

Starting from C# 8.0, you can use expressions in switch cases:


    int number = 2;

    string result = number switch
    {
        1 => "One",
        2 => "Two",
        3 => "Three",
        _ => "Other"
    };

    Console.WriteLine(result);

Explanation

In the provided examples:

  • The first example demonstrates a basic `switch` statement to output the day of the week based on a numeric value.
  • The second example shows how multiple cases can share the same code block to handle grades.
  • The third example uses the switch expression feature introduced in C# 8.0 to return a value based on the case.
  • The `default` case is executed if none of the specified cases match the value of the expression. It is optional but recommended for handling unexpected values.
