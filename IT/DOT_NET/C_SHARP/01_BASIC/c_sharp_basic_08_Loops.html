C# Loops

Loops in C# are used to execute a block of code repeatedly based on a specified condition. They are essential for tasks that require repetition, such as iterating over collections or performing operations multiple times.

1. The `for` Loop

The `for` loop is used when the number of iterations is known beforehand. It consists of three parts: initialization, condition, and iteration.


    for (int i = 0; i < 5; i++)
    {
        Console.WriteLine(i);
    }

This loop initializes i to 0, continues as long as i < 5, and increments i by 1 in each iteration. The output will be:


    0
    1
    2
    3
    4

2. The `while` Loop

The `while` loop is used when the number of iterations is not known beforehand, and the loop continues as long as the condition is true.


    int i = 0;

    while (i < 5)
    {
        Console.WriteLine(i);
        i++;
    }

This loop will output the same values as the `for` loop. It initializes i to 0, checks if i < 5, and increments i inside the loop. The output will be:


    0
    1
    2
    3
    4

3. The `do-while` Loop

The `do-while` loop is similar to the `while` loop, but it guarantees that the block of code will be executed at least once before checking the condition.


    int i = 0;

    do
    {
        Console.WriteLine(i);
        i++;
    }
    while (i < 5);

This loop will also output the same values. The difference is that the loop body is executed before the condition is checked. The output will be:


    0
    1
    2
    3
    4

Hereâ€™s a complete C# program that demonstrates the use of different types of loops:


    using System;

    class Program
    {
        static void Main()
        {
            // for loop
            Console.WriteLine("For Loop:");
            for (int i = 0; i < 3; i++)
            {
                Console.WriteLine(i);
            }

            // while loop
            Console.WriteLine("While Loop:");
            int j = 0;
            while (j < 3)
            {
                Console.WriteLine(j);
                j++;
            }

            // do-while loop
            Console.WriteLine("Do-While Loop:");
            int k = 0;
            do
            {
                Console.WriteLine(k);
                k++;
            }
            while (k < 3);
        }
    }

Explanation

In the example program:

  • The `for` loop iterates three times, printing values from 0 to 2.
  • The `while` loop performs the same iterations as the `for` loop, but with the loop control variables managed separately.
  • The `do-while` loop guarantees that the loop body executes at least once before checking the condition.
