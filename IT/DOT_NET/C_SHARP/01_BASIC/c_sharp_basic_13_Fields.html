C# Fields

In C#, a field is a variable that is declared directly in a class or struct. Fields are used to store data that is associated with a class or object.

1. Defining Fields

Fields are declared inside a class or struct and are typically private to ensure encapsulation. You can also have public fields if you need to expose them directly.


    public class Person
    {
        // Private fields
        private string name;
        private int age;

        // Public fields (less common)
        public string country;
    }

2. Accessing Fields

Fields can be accessed within the class or struct where they are declared. If they are public, they can also be accessed from outside the class. It is generally recommended to use properties for controlled access to fields.


    public class Person
    {
        private string name;
        private int age;

        public Person(string name, int age)
        {
            this.name = name;
            this.age = age;
        }

        public void DisplayInfo()
        {
            Console.WriteLine($"Name: {name}, Age: {age}");
        }
    }

    class Program
    {
        static void Main()
        {
            // Creating an object of the Person class
            Person person = new Person("Alice", 30);
            
            // Accessing private fields via a method
            person.DisplayInfo();
        }
    }

3. Field Modifiers

Fields can be modified with various access modifiers and keywords:

  • private: The field is accessible only within the containing class or struct.
  • public: The field is accessible from any code that can see the class or struct.
  • protected: The field is accessible within the containing class and by derived classes.
  • static: The field belongs to the type itself rather than to any specific instance.
  • readonly: The field can only be assigned during its declaration or in the constructor of the same class.

    public class Person
    {
        // Static field
        public static int populationCount;

        // Read-only field
        public readonly string birthPlace;

        // Private field
        private string name;

        public Person(string name, string birthPlace)
        {
            this.name = name;
            this.birthPlace = birthPlace;
            populationCount++;
        }

        public void DisplayInfo()
        {
            Console.WriteLine($"Name: {name}, BirthPlace: {birthPlace}, Population: {populationCount}");
        }
    }

    class Program
    {
        static void Main()
        {
            // Creating Person objects
            Person person1 = new Person("Alice", "New York");
            Person person2 = new Person("Bob", "Los Angeles");
            
            // Displaying information
            person1.DisplayInfo();
            person2.DisplayInfo();
        }
    }

Explanation

In the example program:

  • The Person class has fields with different modifiers: private, static, and readonly.
  • The populationCount field is static and shared among all instances of the Person class.
  • The birthPlace field is read-only and can only be assigned in the constructor.
  • The DisplayInfo method displays the field values, demonstrating how fields are used in the class.
