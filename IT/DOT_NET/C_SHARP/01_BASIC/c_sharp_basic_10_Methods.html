C# Methods

Methods in C# are blocks of code that perform a specific task. They allow you to organize code into reusable sections, making programs easier to manage and understand.

1. Defining a Method

To define a method, you specify the return type, method name, and optional parameters. The method body is enclosed in curly braces and contains the code to be executed.


    // Method with no parameters and no return value
    void PrintMessage()
    {
        Console.WriteLine("Hello, world!");
    }

    // Method with parameters and a return value
    int Add(int a, int b)
    {
        return a + b;
    }

2. Calling a Method

To call a method, use its name followed by parentheses. If the method has parameters, provide the arguments inside the parentheses.


    // Calling the methods defined above
    class Program
    {
        static void Main()
        {
            // Calling the method with no parameters
            PrintMessage();
            
            // Calling the method with parameters
            int sum = Add(5, 10);
            Console.WriteLine("Sum: " + sum);
        }

        static void PrintMessage()
        {
            Console.WriteLine("Hello, world!");
        }

        static int Add(int a, int b)
        {
            return a + b;
        }
    }

3. Method Overloading

Method overloading allows you to define multiple methods with the same name but different parameters. This can be useful for methods that perform similar tasks but with different types or numbers of inputs.


    // Method Overloading
    class Program
    {
        static void Main()
        {
            Console.WriteLine(Multiply(2, 3));       // Calls the int version
            Console.WriteLine(Multiply(2.5, 3.5));   // Calls the double version
        }

        static int Multiply(int a, int b)
        {
            return a * b;
        }

        static double Multiply(double a, double b)
        {
            return a * b;
        }
    }

4. Passing Parameters by Reference

In C#, you can pass parameters by reference using the ref or out keywords. This allows the method to modify the argument's value directly.


    class Program
    {
        static void Main()
        {
            int value = 10;
            Increment(ref value);
            Console.WriteLine("Value after increment: " + value);
        }

        static void Increment(ref int number)
        {
            number++;
        }
    }

    using System;

        }

        static void PrintMessage()
        {
            Console.WriteLine("Welcome to C# Methods!");
        }

        static int Add(int a, int b)
        {
            return a + b;
        }

        static int Multiply(int a, int b)
        {
            return a * b;
        }

        static double Multiply(double a, double b)
        {
            return a * b;
        }

        static void Increment(ref int number)
        {
            number++;
        }
    }

Explanation

In the example program:

  • The PrintMessage method prints a greeting message.
  • The Add method returns the sum of two integers.
  • The Multiply method is overloaded to handle both integers and doubles.
  • The Increment method demonstrates passing parameters by reference, allowing it to modify the original value.
