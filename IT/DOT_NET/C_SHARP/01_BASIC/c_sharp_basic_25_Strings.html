C# Strings

Strings in C# represent sequences of characters and are one of the most commonly used data types. C# strings are immutable, meaning that once a string is created, it cannot be changed.

2. String Concatenation

Strings can be concatenated using the + operator or the String.Concat method:


    using System;

    class Program
    {
        static void Main()
        {
            string firstName = "John";
            string lastName = "Doe";

            // Concatenate strings
            string fullName = firstName + " " + lastName;

            // Display the concatenated string
            Console.WriteLine(fullName);
        }
    }

3. String Interpolation

String interpolation provides a more readable and convenient syntax for formatting strings. It uses the $ sign before the string literal:


    using System;

    class Program
    {
        static void Main()
        {
            string firstName = "John";
            string lastName = "Doe";

            // String interpolation
            string fullName = $"{firstName} {lastName}";

            // Display the interpolated string
            Console.WriteLine(fullName);
        }
    }

4. Common String Methods

C# provides many useful methods for string manipulation:

  • ToUpper() - Converts the string to uppercase.
  • ToLower() - Converts the string to lowercase.
  • Trim() - Removes leading and trailing whitespace.
  • Substring() - Extracts a substring from the string.
  • Replace() - Replaces all occurrences of a specified substring.

    using System;

    class Program
    {
        static void Main()
        {
            string message = "   Hello, C#!   ";

            // Remove leading and trailing whitespace
            string trimmedMessage = message.Trim();

            // Convert to uppercase
            string upperMessage = trimmedMessage.ToUpper();

            // Replace a substring
            string replacedMessage = upperMessage.Replace("C#", ".NET");

            // Extract a substring
            string subMessage = replacedMessage.Substring(0, 12);

            // Display the results
            Console.WriteLine($"Trimmed: '{trimmedMessage}'");
            Console.WriteLine($"Uppercase: '{upperMessage}'");
            Console.WriteLine($"Replaced: '{replacedMessage}'");
            Console.WriteLine($"Substring: '{subMessage}'");
        }
    }

5. String Comparison

Comparing strings can be done using methods such as Equals(), Compare(), and CompareTo():


    using System;

    class Program
    {
        static void Main()
        {
            string str1 = "apple";
            string str2 = "orange";

            // Compare strings
            bool areEqual = str1.Equals(str2);
            int comparison = string.Compare(str1, str2);

            // Display the results
            Console.WriteLine($"Are strings equal? {areEqual}");
            Console.WriteLine($"Comparison result: {comparison}");
        }
    }

6. Key Points

  • Strings in C# are immutable and represent sequences of characters.
  • String concatenation can be performed using the + operator or String.Concat.
  • String interpolation offers a more readable way to format strings.
  • Common string methods include ToUpper(), ToLower(), Trim(), Substring(), and Replace().
  • String comparison can be done using Equals(), Compare(), and CompareTo().
