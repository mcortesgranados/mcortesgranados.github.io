C# Inheritance

Inheritance is a fundamental concept in object-oriented programming (OOP) that allows a class (child class or subclass) to inherit fields and methods from another class (parent class or superclass). This promotes code reusability and establishes a natural hierarchy between classes.

1. Basics of Inheritance

Inheritance enables a class to use the properties and methods of another class. The class that is inherited from is called the base class, while the class that inherits is called the derived class.


    // Base class
    public class Animal
    {
        public void Eat()
        {
            Console.WriteLine("This animal eats food.");
        }
    }

    // Derived class
    public class Dog : Animal
    {
        public void Bark()
        {
            Console.WriteLine("The dog barks.");
        }
    }

2. Using Inheritance

When creating an instance of a derived class, it automatically inherits all the public and protected members of the base class. You can also add new members to the derived class or override existing ones.


    class Program
    {
        static void Main()
        {
            // Creating an object of the derived class
            Dog myDog = new Dog();
            
            // Calling a method from the base class
            myDog.Eat();
            
            // Calling a method from the derived class
            myDog.Bark();
        }
    }

3. Overriding Methods

Derived classes can override methods from the base class to provide specific implementations. The base class method must be marked as virtual, and the derived class method must use the override keyword.


    // Base class
    public class Animal
    {
        public virtual void MakeSound()
        {
            Console.WriteLine("This animal makes a sound.");
        }
    }

    // Derived class
    public class Dog : Animal
    {
        public override void MakeSound()
        {
            Console.WriteLine("The dog barks.");
        }
    }

5. Key Points

  • Inheritance allows a class to inherit members from another class, facilitating code reuse.
  • Base Class is the class being inherited from.
  • Derived Class is the class that inherits from the base class.
  • Method Overriding lets a derived class provide a specific implementation of a method defined in the base class.
  • Use virtual in the base class method and override in the derived class method to override functionality.
