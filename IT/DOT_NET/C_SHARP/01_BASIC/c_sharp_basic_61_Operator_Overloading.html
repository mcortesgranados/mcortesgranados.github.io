C# Operator Overloading

Operator overloading in C# allows you to define custom behaviors for operators when applied to your user-defined types. This can make your classes and structs work more naturally with operators, improving readability and ease of use.

1. What is Operator Overloading?

Operator overloading is the process of defining how operators (such as +, -, *, /) work with instances of your classes or structs. By overloading operators, you can specify how these operators should behave when applied to your types.

2. How to Overload Operators

To overload an operator, you need to define a static method in your class or struct with the operator keyword, followed by the operator symbol you want to overload. The method must have the correct signature to match the operator's requirements.

3. Example

Here's an example of overloading the + operator for a Point class to add two points together:


public class Point
{
    public int X { get; set; }
    public int Y { get; set; }

    public Point(int x, int y)
    {
        X = x;
        Y = y;
    }

    // Overload the + operator
    public static Point operator +(Point p1, Point p2)
    {
        return new Point(p1.X + p2.X, p1.Y + p2.Y);
    }

    // Overload the - operator
    public static Point operator -(Point p1, Point p2)
    {
        return new Point(p1.X - p2.X, p1.Y - p2.Y);
    }

    public override string ToString()
    {
        return $"({X}, {Y})";
    }
}

In this example, we have overloaded the + and - operators for the Point class. Now, you can use these operators to add and subtract points:


Point p1 = new Point(2, 3);
Point p2 = new Point(5, 7);
Point resultAdd = p1 + p2; // Uses overloaded + operator
Point resultSub = p1 - p2; // Uses overloaded - operator

Console.WriteLine(resultAdd); // Output: (7, 10)
Console.WriteLine(resultSub); // Output: (-3, -4)

4. Key Points

  • Definition: Overload operators to define custom behaviors for operators with your types.
  • Syntax: Use the operator keyword followed by the operator symbol in a static method.
  • Example: Overloading + and - operators for a Point class to add and subtract points.
  • Usage: Overloading operators can make your types work more naturally with operators, improving code readability.
