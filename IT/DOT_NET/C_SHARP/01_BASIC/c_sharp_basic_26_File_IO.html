C# File I/O

File I/O (Input/Output) in C# allows you to read from and write to files. C# provides several classes in the System.IO namespace to handle file operations.

1. Writing to a File

You can write text to a file using the StreamWriter class. Hereâ€™s how to create and write to a file:


    using System;
    using System.IO;

    class Program
    {
        static void Main()
        {
            // Define the file path
            string filePath = "example.txt";

            // Create and write to the file
            using (StreamWriter writer = new StreamWriter(filePath))
            {
                writer.WriteLine("Hello, File!");
                writer.WriteLine("This is a sample file.");
            }

            Console.WriteLine("File written successfully.");
        }
    }

2. Reading from a File

To read text from a file, you can use the StreamReader class:


    using System;
    using System.IO;

    class Program
    {
        static void Main()
        {
            // Define the file path
            string filePath = "example.txt";

            // Read from the file
            using (StreamReader reader = new StreamReader(filePath))
            {
                string content = reader.ReadToEnd();
                Console.WriteLine("File content:");
                Console.WriteLine(content);
            }
        }
    }

3. Checking File Existence

Before performing file operations, it is good practice to check if the file exists using File.Exists():


    using System;
    using System.IO;

    class Program
    {
        static void Main()
        {
            string filePath = "example.txt";

            if (File.Exists(filePath))
            {
                Console.WriteLine("File exists.");
            }
            else
            {
                Console.WriteLine("File does not exist.");
            }
        }
    }

4. Appending to a File

If you want to add data to an existing file without overwriting its content, you can use the StreamWriter with the append parameter set to true:


    using System;
    using System.IO;

    class Program
    {
        static void Main()
        {
            string filePath = "example.txt";

            // Append to the file
            using (StreamWriter writer = new StreamWriter(filePath, true))
            {
                writer.WriteLine("Appending new line to the file.");
            }

            Console.WriteLine("File appended successfully.");
        }
    }

5. Key Points

  • StreamWriter is used for writing text to files.
  • StreamReader is used for reading text from files.
  • File.Exists() checks if a file exists before performing operations.
  • Appending data to a file can be done by passing true to the StreamWriter constructor.
