Understanding C# Variables

In C#, variables are used to store data that can be manipulated and used throughout a program. Each variable in C# has a type that determines the kind of data it can hold.

1. Declaring Variables

To declare a variable, you specify its type followed by its name. For example:

int age;
string name;

2. Initializing Variables

Once a variable is declared, you can assign it a value. This is called initializing a variable:

age = 30;
name = "Alice";

3. Variable Types

C# supports various types of variables, including:

  • int: Represents integer values (e.g., 10, -3).
  • double: Represents floating-point numbers (e.g., 3.14, -0.001).
  • char: Represents a single character (e.g., 'A', 'b').
  • string: Represents a sequence of characters (e.g., "Hello, World!").
  • bool: Represents a boolean value (e.g., true, false).

4. Example Program

Here's a simple C# program that demonstrates the use of variables:


    using System;

    class Program
    {
        static void Main()
        {
            int age = 30;
            string name = "Alice";
            double height = 5.7;
            bool isStudent = true;

            Console.WriteLine("Name: " + name);
            Console.WriteLine("Age: " + age);
            Console.WriteLine("Height: " + height);
            Console.WriteLine("Is a student: " + isStudent);
        }
    }

Explanation

In the example program:

  • int age = 30;: Declares an integer variable named age and initializes it with the value 30.
  • string name = "Alice";: Declares a string variable named name and initializes it with the value "Alice".
  • double height = 5.7;: Declares a double variable named height and initializes it with the value 5.7.
  • bool isStudent = true;: Declares a boolean variable named isStudent and initializes it with the value true.
  • Console.WriteLine(...);: Outputs the values of the variables to the console.
