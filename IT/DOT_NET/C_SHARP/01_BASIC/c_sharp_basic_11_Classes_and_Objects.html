C# Classes and Objects

In C#, a class is a blueprint for creating objects. It defines properties, methods, and other members that the objects created from the class can use. An object is an instance of a class.

1. Defining a Class

A class is defined using the class keyword followed by the class name. It can include fields (variables), properties, methods, and constructors.


    // Defining a class
    public class Person
    {
        // Fields
        public string Name;
        public int Age;

        // Constructor
        public Person(string name, int age)
        {
            Name = name;
            Age = age;
        }

        // Method
        public void Introduce()
        {
            Console.WriteLine($"Hello, my name is {Name} and I am {Age} years old.");
        }
    }

2. Creating and Using Objects

To create an object, use the new keyword followed by the class constructor. Once an object is created, you can access its members using the dot notation.


    class Program
    {
        static void Main()
        {
            // Creating an object of the Person class
            Person person = new Person("Alice", 30);
            
            // Calling the method of the Person class
            person.Introduce();

            // Accessing fields
            Console.WriteLine($"Name: {person.Name}");
            Console.WriteLine($"Age: {person.Age}");
        }
    }

3. Properties

Properties provide a way to read, write, or compute the values of private fields. They are defined using get and set accessors.


    public class Person
    {
        private string name;
        private int age;

        // Property
        public string Name
        {
            get { return name; }
            set { name = value; }
        }

        public int Age
        {
            get { return age; }
            set
            {
                if (value > 0)
                    age = value;
                else
                    Console.WriteLine("Age must be positive.");
            }
        }

        // Constructor
        public Person(string name, int age)
        {
            Name = name;
            Age = age;
        }

        // Method
        public void Introduce()
        {
            Console.WriteLine($"Hello, my name is {Name} and I am {Age} years old.");
        }
    }

4. Inheritance

Inheritance allows a class to inherit members from another class. The base class is the class being inherited from, and the derived class is the class that inherits.


    // Base class
    public class Animal
    {
        public void Eat()
        {
            Console.WriteLine("Eating...");
        }
    }

    // Derived class
    public class Dog : Animal
    {
        public void Bark()
        {
            Console.WriteLine("Woof!");
        }
    }

    class Program
    {
        static void Main()
        {
            // Creating an object of the Dog class
            Dog dog = new Dog();
            
            // Calling methods from both the base and derived classes
            dog.Eat();
            dog.Bark();
        }
    }

    }

    class Program

Explanation

In the example program:

  • The Main method creates objects of the Person and Dog classes and calls their methods to illustrate their functionality.
