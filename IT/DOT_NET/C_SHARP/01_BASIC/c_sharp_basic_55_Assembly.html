C# Assemblies

In C#, an assembly is a fundamental building block of .NET applications. It is a compiled code library used by .NET applications and provides the runtime environment with metadata about the application.

1. Purpose

Assemblies serve several purposes in a .NET application:

  • Modularization: Divide code into manageable and reusable units.
  • Versioning: Manage different versions of code components.
  • Deployment: Package and deploy application components.
  • Security: Enforce code access security through strong naming.

2. Types of Assemblies

There are two main types of assemblies:

  • Private Assemblies: Used by a single application and are stored in the application's directory.
  • Shared Assemblies: Can be used by multiple applications and are usually stored in the Global Assembly Cache (GAC).

Assemblies are created by compiling C# code. Hereâ€™s an example of how to create a simple class library assembly:


// Example: MyLibrary.cs

using System;

namespace MyLibrary
{
    public class Greeter
    {
        public string Greet(string name)
        {
            return $"Hello, {name}!";
        }
    }
}

To compile this code into an assembly, use the C# compiler (csc) or an IDE like Visual Studio to build the project.

4. Assembly Metadata

Assemblies contain metadata about the types they contain. This metadata includes information such as:

  • Assembly Name: The name of the assembly.
  • Version: The version number of the assembly.
  • Culture: The culture/locale the assembly supports.
  • Public Key: If the assembly is strongly named, the public key used to sign it.

Metadata is used by the .NET runtime for tasks such as loading and linking assemblies and enforcing security.

5. Using Assemblies

To use an assembly, reference it in your project. For example, if you have created a class library assembly named MyLibrary.dll, you can reference it in another project:


// Example: Program.cs

using System;
using MyLibrary;  // Reference to the MyLibrary assembly

class Program
{
    static void Main()
    {
        Greeter greeter = new Greeter();
        Console.WriteLine(greeter.Greet("World"));
    }
}
