C# Directory Operations

Directory operations in C# allow you to manage directories by creating, deleting, and listing them. These operations are performed using classes from the System.IO namespace.

1. Creating a Directory

You can create a new directory using the Directory.CreateDirectory() method:


    using System;
    using System.IO;

    class Program
    {
        static void Main()
        {
            // Define the directory path
            string directoryPath = "new_directory";

            // Create the directory
            Directory.CreateDirectory(directoryPath);

            Console.WriteLine("Directory created successfully.");
        }
    }

2. Deleting a Directory

To delete a directory, use the Directory.Delete() method. By default, this method only works if the directory is empty. Use the overload with the recursive parameter to delete non-empty directories:


    using System;
    using System.IO;

    class Program
    {
        static void Main()
        {
            // Define the directory path
            string directoryPath = "new_directory";

            // Delete the directory (true for recursive deletion)
            Directory.Delete(directoryPath, true);

            Console.WriteLine("Directory deleted successfully.");
        }
    }

3. Listing Directories

You can list directories in a specified path using the Directory.GetDirectories() method:


    using System;
    using System.IO;

    class Program
    {
        static void Main()
        {
            // Define the path to list directories
            string path = "."; // Current directory

            // Get directories
            string[] directories = Directory.GetDirectories(path);

            Console.WriteLine("Directories:");
            foreach (string dir in directories)
            {
                Console.WriteLine(dir);
            }
        }
    }

4. Checking Directory Existence

Before performing operations on a directory, it is good practice to check if the directory exists using Directory.Exists():


    using System;
    using System.IO;

    class Program
    {
        static void Main()
        {
            string directoryPath = "new_directory";

            if (Directory.Exists(directoryPath))
            {
                Console.WriteLine("Directory exists.");
            }
            else
            {
                Console.WriteLine("Directory does not exist.");
            }
        }
    }

5. Key Points

  • Directory.CreateDirectory() is used to create a new directory.
  • Directory.Delete() is used to delete a directory. Use the recursive overload to delete non-empty directories.
  • Directory.GetDirectories() lists all directories in a specified path.
  • Directory.Exists() checks if a directory exists before performing operations.
