Writing Your First C# Program

Creating your first C# program is a great way to get started with the language. In this example, we'll write a simple "Hello, World!" program.

1. Open Visual Studio

Launch Visual Studio and create a new C# project. You can do this by selecting File > New > Project from the menu.

2. Choose a Project Template

Select the "Console App (.NET Core)" or "Console Application" template. This will create a simple application that runs in the command line.

3. Write Your First Program

Replace the existing code in Program.cs with the following:


    using System;

    class Program
    {
        static void Main()
        {
            Console.WriteLine("Hello, World!");
        }
    }

4. Run Your Program

To run your program, press F5 or click the Start button in Visual Studio. You should see the output:

Hello, World!

Explanation

Here's a brief overview of the code:

  • using System;: Imports the System namespace, which provides basic functionality including the Console class.
  • class Program: Defines a class named Program, which is the main entry point of the application.
  • static void Main(): Defines the Main method, which is the starting point of the program. It does not return any value.
  • Console.WriteLine("Hello, World!");: Outputs the text "Hello, World!" to the console.
