C# Arrays

An array in C# is a data structure that can hold a fixed number of elements of the same type. Arrays are useful for storing multiple values in a single variable, and they are indexed, which means you can access each element by its index.

1. Declaring and Initializing Arrays

To declare an array, specify the type of elements it will hold, followed by square brackets. You can initialize an array at the time of declaration or later.

Declaration and Initialization


    // Declaring and initializing an array in one line
    int[] numbers = { 1, 2, 3, 4, 5 };
    
    // Declaring an array and then initializing it
    int[] moreNumbers;
    moreNumbers = new int[] { 6, 7, 8, 9, 10 };

2. Accessing Array Elements

Array elements are accessed using their index. In C#, array indices start at 0.


    int[] numbers = { 1, 2, 3, 4, 5 };

    // Accessing elements
    Console.WriteLine(numbers[0]); // Outputs: 1
    Console.WriteLine(numbers[3]); // Outputs: 4

3. Iterating Through Arrays

You can use a `for` loop or a `foreach` loop to iterate through the elements of an array.

Using a `for` Loop


    int[] numbers = { 1, 2, 3, 4, 5 };

    for (int i = 0; i < numbers.Length; i++)
    {
        Console.WriteLine(numbers[i]);
    }

Using a `foreach` Loop


    int[] numbers = { 1, 2, 3, 4, 5 };

    foreach (int number in numbers)
    {
        Console.WriteLine(number);
    }

4. Multi-Dimensional Arrays

C# supports multi-dimensional arrays, such as two-dimensional arrays, which can be thought of as a table or grid.


    // Declaring and initializing a 2D array
    int[,] matrix = {
        { 1, 2, 3 },
        { 4, 5, 6 },
        { 7, 8, 9 }
    };

    // Accessing elements
    Console.WriteLine(matrix[0, 0]); // Outputs: 1
    Console.WriteLine(matrix[1, 2]); // Outputs: 6

    using System;

    class Program
    {
        static void Main()
        {
            // 1D Array
            int[] numbers = { 1, 2, 3, 4, 5 };
            Console.WriteLine("1D Array:");
            foreach (int number in numbers)
            {
                Console.WriteLine(number);
            }

            // 2D Array
            int[,] matrix = {
                { 1, 2, 3 },
                { 4, 5, 6 },
                { 7, 8, 9 }
            };
            Console.WriteLine("\n2D Array:");
            for (int i = 0; i < matrix.GetLength(0); i++)
            {
                for (int j = 0; j < matrix.GetLength(1); j++)
                {
                    Console.Write(matrix[i, j] + " ");
                }
                Console.WriteLine();
            }
        }
    }

Explanation

In the example program:

  • The first part demonstrates a one-dimensional array and iterates through it using a `foreach` loop.
  • The second part shows a two-dimensional array and uses nested `for` loops to iterate through rows and columns, printing the values in a matrix format.
