C# Enumerations

Enumerations, or enums, in C# are a special "class" that represents a group of named integral constants. They are used to define a variable that can hold a set of predefined values. Enums make the code more readable and maintainable by giving names to integral values.

1. Purpose

Enumerations are used for:

  • Improving Code Readability: Use meaningful names for sets of related constants.
  • Maintaining Code: Provide a way to group related constants in a type-safe manner.
  • Reducing Errors: Limit the values that a variable can hold, reducing the chance of invalid values.

2. Declaring an Enumeration

To declare an enum, use the enum keyword followed by the enum name and a set of named constants.


    public enum DaysOfWeek
    {
        Sunday,
        Monday,
        Tuesday,
        Wednesday,
        Thursday,
        Friday,
        Saturday
    }

3. Using an Enumeration

Enumerations can be used in code to represent and compare values.


    using System;

    public enum DaysOfWeek
    {
        Sunday,
        Monday,
        Tuesday,
        Wednesday,
        Thursday,
        Friday,
        Saturday
    }

    class Program
    {
        static void Main()
        {
            DaysOfWeek today = DaysOfWeek.Wednesday;

            if (today == DaysOfWeek.Wednesday)
            {
                Console.WriteLine("It's hump day!");
            }
        }
    }

4. Assigning Specific Values

By default, the first enumerator has the value 0, and the value of each successive enumerator is increased by 1. You can assign specific values to enumerators if needed.


    public enum DaysOfWeek
    {
        Sunday = 1,
        Monday = 2,
        Tuesday = 3,
        Wednesday = 4,
        Thursday = 5,
        Friday = 6,
        Saturday = 7
    }

5. Using Enum with Switch Statements

Enums are commonly used in switch statements to handle different cases based on the enum values.


    using System;

    public enum DaysOfWeek
    {
        Sunday,
        Monday,
        Tuesday,
        Wednesday,
        Thursday,
        Friday,
        Saturday
    }

    class Program
    {
        static void Main()
        {
            DaysOfWeek today = DaysOfWeek.Friday;

            switch (today)
            {
                case DaysOfWeek.Monday:
                    Console.WriteLine("Start of the work week.");
                    break;
                case DaysOfWeek.Friday:
                    Console.WriteLine("End of the work week.");
                    break;
                case DaysOfWeek.Saturday:
                case DaysOfWeek.Sunday:
                    Console.WriteLine("Weekend!");
                    break;
                default:
                    Console.WriteLine("Midweek.");
                    break;
            }
        }
    }

6. Enum and Underlying Types

By default, enums use the int type as their underlying type. You can specify a different integral type if needed.


    public enum DaysOfWeek : byte
    {
        Sunday = 1,
        Monday = 2,
        Tuesday = 3,
        Wednesday = 4,
        Thursday = 5,
        Friday = 6,
        Saturday = 7
    }

7. Key Points

  • Enums define a set of named integral constants.
  • Improves code readability and maintainability.
  • Enums can be used in switch statements for clear and structured code.
  • Underlying type can be customized if necessary.
