C# Null-Coalescing Operator

The null-coalescing operator in C# provides a way to define a default value that will be used if an expression evaluates to null. This operator simplifies code that checks for null values and provides a fallback option.

1. What is the Null-Coalescing Operator?

The null-coalescing operator is represented by ??. It allows you to specify a default value to use when an expression evaluates to null.

2. How to Use the Null-Coalescing Operator

When you use the ?? operator, it evaluates the expression on its left side. If that expression is not null, its value is returned. If it is null, the value on the right side of the operator is returned.

3. Example

Hereâ€™s an example demonstrating the use of the null-coalescing operator:


string name = null;
string defaultName = "Guest";

// Use the null-coalescing operator to provide a default value
string displayName = name ?? defaultName;

Console.WriteLine(displayName); // Output: Guest

In this example, the name variable is null, so the defaultName is used instead.

4. Null-Coalescing Assignment Operator

C# 8.0 introduced the null-coalescing assignment operator ??=. This operator assigns a value to a variable only if that variable is null.


string name = null;
name ??= "Guest";

Console.WriteLine(name); // Output: Guest

In this example, the name variable is assigned the value "Guest" only because it was initially null.

5. Key Points

  • Definition: The null-coalescing operator ?? provides a default value when an expression is null.
  • Usage: Simplifies code by reducing the need for explicit null checks and conditionals.
  • Null-Coalescing Assignment: The ??= operator assigns a value only if the variable is null.
