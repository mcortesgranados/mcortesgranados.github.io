Oracle Forms Q&A - Part 2

1. What is a parameter form in Oracle Forms?

A parameter form is used to collect input parameters from the user before running a report or other operation. It typically includes fields for the user to enter values.

2. How can you use a parameter form to filter data in a report?

When a parameter form is submitted, the values entered by the user can be passed to a query or a report to filter the data accordingly.

3. What is the difference between WHEN-NEW-RECORD-INSTANCE and WHEN-NEW-FORM-INSTANCE triggers?

WHEN-NEW-RECORD-INSTANCE is triggered when a new record is displayed or created, while WHEN-NEW-FORM-INSTANCE is triggered when the form is first loaded.

4. How can you create a custom menu in Oracle Forms?

You can create a custom menu using the Menu Editor in Oracle Forms Builder. Define menu items and associate them with triggers or procedures.

5. What is a visual attribute in Oracle Forms?

Visual attributes are properties such as color, font, and style that define the appearance of items in Oracle Forms.

6. How do you change the appearance of a field in Oracle Forms?

You can change the appearance by modifying its visual attributes through the property palette or by creating and applying a custom visual attribute.

7. How can you handle multi-record blocks in Oracle Forms?

Multi-record blocks can be handled by using LOVs or programmatically looping through records using PL/SQL to perform operations on each record.

8. What is a key trigger in Oracle Forms?

A key trigger is associated with a specific keyboard action, such as pressing a key or key combination. It allows you to define custom behavior for keyboard inputs.

9. How do you use the SET_ITEM_PROPERTY built-in procedure?

The SET_ITEM_PROPERTY procedure is used to change properties of form items dynamically at runtime.


SET_ITEM_PROPERTY('item_name', VISUAL_ATTRIBUTE, 'new_attribute');
10. What are Oracle Forms libraries and how are they used?

Libraries in Oracle Forms are PL/SQL code modules that can be shared across multiple forms. They are used to store reusable procedures and functions.

11. How do you debug a form in Oracle Forms?

You can debug a form by using the built-in debugger in Oracle Forms Builder, setting breakpoints, and examining variable values during execution.

12. What is the purpose of the WHEN-MOUSE-CLICK trigger?

The WHEN-MOUSE-CLICK trigger is used to handle mouse click events on items or canvases, allowing you to execute custom code in response to user clicks.

13. How can you enable or disable a field dynamically?

You can enable or disable a field using the SET_ITEM_PROPERTY procedure to change its ENABLED property.


SET_ITEM_PROPERTY('item_name', ENABLED, PROPERTY_TRUE);
14. What is the WHEN-CHECK-BOX-CHANGED trigger used for?

The WHEN-CHECK-BOX-CHANGED trigger is used to handle changes in the state of a checkbox, allowing you to execute code when the checkbox is checked or unchecked.

15. How do you perform a database operation using Oracle Forms?

You can perform database operations such as queries, inserts, updates, and deletes using built-in PL/SQL procedures like EXECUTE_QUERY, INSERT_RECORD, and DELETE_RECORD.

16. What is the purpose of the QUERY_DATA built-in procedure?

The QUERY_DATA procedure is used to execute a query and populate a data block with the results.


QUERY_DATA;
17. How do you create a tab canvas in Oracle Forms?

A tab canvas is created by using the Canvas Editor to define tab pages and then adding items to these pages. You set up navigation between the tabs using triggers.

18. What is the purpose of the POST built-in procedure?

The POST procedure is used to write changes made to the current form's data block to the database without committing them.


POST;
19. How do you create a report in Oracle Forms?

Reports can be created using Oracle Reports Builder, and then integrated into Oracle Forms using RUN_REPORT_OBJECT to generate and display the report.

20. What are the types of canvases available in Oracle Forms?

The main types of canvases are Content Canvases, Stack Canvases, and Tab Canvases. Each serves different purposes for layout and item organization.
