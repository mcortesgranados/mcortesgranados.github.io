Python Basic Questions and Answers

1. What is Python?

Python is an interpreted, high-level, and dynamically typed programming language that emphasizes code readability.

2. How do you print text in Python?

Use the print() function to display text.

print("Hello, World!")

3. How do you take user input in Python?

Use the input() function.

name = input("Enter your name: ")
print("Hello, " + name)

4. What is a variable in Python?

A variable is a name that refers to a value stored in memory.

x = 10
y = "Python"
print(x, y)

5. How do you perform basic arithmetic operations?

Use operators like +, -, *, /, //, and %.

a = 10
b = 5
print(a + b, a - b, a * b, a / b, a // b, a % b)

6. What is the difference between == and is in Python?

== checks for value equality, whereas is checks for identity (memory location).

a = [1, 2, 3]
b = [1, 2, 3]
print(a == b)  # True (same values)
print(a is b)  # False (different objects in memory)

7. How do you create a list in Python?

Use square brackets [] to define a list.

fruits = ["apple", "banana", "cherry"]
print(fruits)

8. How do you access elements in a list?

Use indexing. Python uses zero-based indexing.

fruits = ["apple", "banana", "cherry"]
print(fruits[0])  # apple
print(fruits[-1])  # cherry

9. How do you iterate over a list?

Use a for loop.

fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)

10. How do you check if an item exists in a list?

Use the in keyword.

fruits = ["apple", "banana", "cherry"]
print("banana" in fruits)  # True
print("grape" in fruits)  # False

11. What is a tuple in Python?

A tuple is an immutable ordered collection of elements, defined using parentheses ().

my_tuple = (1, 2, 3, "hello")
print(my_tuple)

12. How do you access elements in a tuple?

Use indexing like lists.

my_tuple = (1, 2, 3, "hello")
print(my_tuple[0])  # 1
print(my_tuple[-1])  # "hello"

13. What is the difference between a list and a tuple?

Lists are mutable (can be changed), while tuples are immutable (cannot be changed).

my_list = [1, 2, 3]
my_list[0] = 100  # Allowed

my_tuple = (1, 2, 3)
# my_tuple[0] = 100  # Error: Tuples are immutable

14. How do you create a dictionary in Python?

Use curly braces {} with key-value pairs.

person = {"name": "Alice", "age": 25, "city": "New York"}
print(person)

15. How do you access values in a dictionary?

Use keys to access values.

person = {"name": "Alice", "age": 25, "city": "New York"}
print(person["name"])  # Alice
print(person.get("age"))  # 25

16. How do you add or update values in a dictionary?

Use key assignment.

person = {"name": "Alice", "age": 25}
person["city"] = "New York"  # Add new key-value pair
person["age"] = 26  # Update value
print(person)

17. How do you remove a key from a dictionary?

Use the del statement or the pop() method.

person = {"name": "Alice", "age": 25, "city": "New York"}
del person["age"]
print(person)

city = person.pop("city")
print(city)  # New York

18. How do you iterate over dictionary keys and values?

Use the items() method.

person = {"name": "Alice", "age": 25, "city": "New York"}
for key, value in person.items():
    print(key, ":", value)

19. What are Python sets?

A set is an unordered collection of unique elements.

my_set = {1, 2, 3, 3, 2, 1}
print(my_set)  # {1, 2, 3}

20. How do you check if an item exists in a set?

Use the in keyword.

my_set = {1, 2, 3}
print(2 in my_set)  # True
print(5 in my_set)  # False

21. How do you add and remove elements from a set?

Use the add() and remove() methods.

my_set = {1, 2, 3}
my_set.add(4)
print(my_set)  # {1, 2, 3, 4}

my_set.remove(2)
print(my_set)  # {1, 3, 4}

22. What are Python conditional statements?

Conditional statements control flow using if, elif, and else.

x = 10
if x > 0:
    print("Positive")
elif x < 0:
    print("Negative")
else:
    print("Zero")

23. What is a loop in Python?

Loops allow repeated execution of a block of code. Python supports for and while loops.

24. How do you use a for loop in Python?

Use for with an iterable like a list.

numbers = [1, 2, 3]
for num in numbers:
    print(num)

25. How do you use a while loop in Python?

Use while with a condition.

x = 5
while x > 0:
    print(x)
    x -= 1

26. How do you exit a loop prematurely?

Use the break statement.

for i in range(10):
    if i == 5:
        break
    print(i)

27. How do you skip an iteration in a loop?

Use the continue statement.

for i in range(5):
    if i == 2:
        continue
    print(i)

28. What is a function in Python?

A function is a reusable block of code.

def greet(name):
    return "Hello, " + name

print(greet("Alice"))

29. How do you define a function with default parameters?

Use an equal sign = in the parameter list.

def greet(name="Guest"):
    return "Hello, " + name

print(greet())  # Hello, Guest
print(greet("Alice"))  # Hello, Alice

30. What is the difference between return and print() in a function?

return gives back a value, while print() only displays output.

def add(a, b):
    return a + b  # Returns the sum

result = add(3, 4)
print(result)  # Displays 7

31. What is a lambda function in Python?

A lambda function is an anonymous function defined using the lambda keyword.

add = lambda x, y: x + y
print(add(3, 4))  # 7

32. How do you use a lambda function inside map()?

map() applies a function to all items in an iterable.

numbers = [1, 2, 3, 4]
squared = list(map(lambda x: x**2, numbers))
print(squared)  # [1, 4, 9, 16]

33. How do you use filter() with a lambda function?

filter() selects elements that satisfy a condition.

numbers = [1, 2, 3, 4, 5]
even_numbers = list(filter(lambda x: x % 2 == 0, numbers))
print(even_numbers)  # [2, 4]

34. How do you use the reduce() function?

reduce() applies a function cumulatively to elements of an iterable.

from functools import reduce

numbers = [1, 2, 3, 4]
product = reduce(lambda x, y: x * y, numbers)
print(product)  # 24

35. What is list comprehension in Python?

List comprehension provides a concise way to create lists.

numbers = [1, 2, 3, 4]
squared = [x**2 for x in numbers]
print(squared)  # [1, 4, 9, 16]

36. How do you use list comprehension with a condition?

You can filter elements using an if statement.

numbers = [1, 2, 3, 4, 5, 6]
even_numbers = [x for x in numbers if x % 2 == 0]
print(even_numbers)  # [2, 4, 6]

37. What is dictionary comprehension?

It allows creating dictionaries in a single line.

numbers = [1, 2, 3, 4]
squares = {x: x**2 for x in numbers}
print(squares)  # {1: 1, 2: 4, 3: 9, 4: 16}

38. What is a Python module?

A module is a file containing Python code, usually functions or classes.

# mymodule.py
def greet(name):
    return "Hello, " + name

39. How do you import a module?

Use the import statement.

import mymodule
print(mymodule.greet("Alice"))

40. How do you import specific functions from a module?

Use from ... import ....

from mymodule import greet
print(greet("Alice"))

41. What is the difference between import module and from module import *?

import module requires you to prefix functions with the module name, while from module import * imports everything directly.

# Using import module
import math
print(math.sqrt(16))  # 4.0

# Using from module import *
from math import *
print(sqrt(16))  # 4.0

A package is a collection of modules organized in directories containing an __init__.py file.

# Directory structure:
# mypackage/
# ├── __init__.py
# ├── module1.py
# ├── module2.py

43. How do you install external Python packages?

Use the pip command.

pip install requests

44. How do you read a file in Python?

Use the open() function.

with open("file.txt", "r") as file:
    content = file.read()
print(content)

45. How do you write to a file in Python?

Use the write() method.

with open("file.txt", "w") as file:
    file.write("Hello, world!")

46. How do you append to a file?

Use mode "a" (append).

with open("file.txt", "a") as file:
    file.write("\nAppending new content.")

47. How do you read a file line by line?

Use a loop over readlines().

with open("file.txt", "r") as file:
    for line in file:
        print(line.strip())

48. What is exception handling in Python?

It allows handling runtime errors using try and except.

try:
    x = 10 / 0
except ZeroDivisionError:
    print("Cannot divide by zero")

49. How do you handle multiple exceptions?

Use multiple except blocks.

try:
    x = int("hello")
except ValueError:
    print("Invalid value")
except ZeroDivisionError:
    print("Cannot divide by zero")

50. How do you use the finally block?

The finally block always executes, whether an exception occurs or not.

try:
    x = 10 / 0
except ZeroDivisionError:
    print("Error occurred")
finally:
    print("This will always execute")

51. What is the purpose of the raise keyword?

The raise keyword is used to manually trigger an exception.

def divide(a, b):
    if b == 0:
        raise ValueError("Denominator cannot be zero")
    return a / b

try:
    result = divide(10, 0)
except ValueError as e:
    print(e)  # Denominator cannot be zero

52. What is the assert statement used for?

assert is used for debugging by checking if a condition is true.

x = 10
assert x > 5  # No error
assert x < 5, "x is not less than 5"  # Raises AssertionError

53. What is a class in Python?

A class is a blueprint for creating objects.

class Car:
    def __init__(self, brand):
        self.brand = brand

    def honk(self):
        return "Beep!"

my_car = Car("Toyota")
print(my_car.brand)  # Toyota
print(my_car.honk())  # Beep!

54. What is an object in Python?

An object is an instance of a class.

class Dog:
    def __init__(self, name):
        self.name = name

dog1 = Dog("Buddy")
print(dog1.name)  # Buddy

55. What is inheritance in Python?

Inheritance allows one class to inherit attributes and methods from another.

class Animal:
    def speak(self):
        return "Some sound"

class Dog(Animal):
    def speak(self):
        return "Woof!"

dog = Dog()
print(dog.speak())  # Woof!

56. What is method overriding?

Method overriding allows a subclass to provide a different implementation of a method from its parent class.

class Parent:
    def show(self):
        return "Parent method"

class Child(Parent):
    def show(self):
        return "Child method"

c = Child()
print(c.show())  # Child method

57. What is method overloading in Python?

Python does not support method overloading natively, but it can be implemented using default arguments.

class Math:
    def add(self, a, b, c=0):
        return a + b + c

m = Math()
print(m.add(2, 3))     # 5
print(m.add(2, 3, 4))  # 9

58. What is multiple inheritance?

Multiple inheritance allows a class to inherit from more than one parent class.

class A:
    def method_a(self):
        return "A"

class B:
    def method_b(self):
        return "B"

class C(A, B):
    pass

obj = C()
print(obj.method_a())  # A
print(obj.method_b())  # B

59. What is a static method in Python?

A static method does not depend on an instance and is defined using @staticmethod.

class Utility:
    @staticmethod
    def add(x, y):
        return x + y

print(Utility.add(3, 5))  # 8

60. What is a class method in Python?

A class method operates on the class rather than an instance and is defined using @classmethod.

class Example:
    count = 0

    @classmethod
    def increment(cls):
        cls.count += 1
        return cls.count

print(Example.increment())  # 1
print(Example.increment())  # 2

61. What is the difference between a class method and a static method?

A class method operates on the class and takes cls as its first argument, while a static method does not depend on the class or instance.

class Example:
    value = 10

    @classmethod
    def class_method(cls):
        return cls.value

    @staticmethod
    def static_method():
        return "I do not use class or instance"

print(Example.class_method())  # 10
print(Example.static_method())  # I do not use class or instance

62. What is the difference between is and == in Python?

is checks if two objects reference the same memory location, while == checks for value equality.

a = [1, 2, 3]
b = a
c = [1, 2, 3]

print(a == c)  # True (same values)
print(a is c)  # False (different objects)
print(a is b)  # True (same object)

63. What is the self keyword in Python?

self represents the instance of the class and is used to access instance attributes and methods.

class Person:
    def __init__(self, name):
        self.name = name

    def greet(self):
        return f"Hello, my name is {self.name}"

p = Person("Alice")
print(p.greet())  # Hello, my name is Alice

64. What is the difference between deep copy and shallow copy?

A shallow copy creates a new object but copies references to the original objects, while a deep copy creates a completely independent copy.

import copy

original = [[1, 2, 3], [4, 5, 6]]
shallow = copy.copy(original)
deep = copy.deepcopy(original)

original[0][0] = 99
print(shallow[0][0])  # 99 (affected)
print(deep[0][0])  # 1 (not affected)

65. What is the purpose of the super() function?

super() allows you to call methods from a parent class inside a subclass.

class Parent:
    def show(self):
        return "Parent method"

class Child(Parent):
    def show(self):
        return super().show() + " and Child method"

c = Child()
print(c.show())  # Parent method and Child method

66. What is the property decorator used for?

It allows defining getter and setter methods in a class.

class Person:
    def __init__(self, age):
        self._age = age

    @property
    def age(self):
        return self._age

    @age.setter
    def age(self, value):
        if value < 0:
            raise ValueError("Age cannot be negative")
        self._age = value

p = Person(25)
print(p.age)  # 25
p.age = 30
print(p.age)  # 30

67. What is an abstract class in Python?

An abstract class is a class that cannot be instantiated and must be subclassed, using the abc module.

from abc import ABC, abstractmethod

class Animal(ABC):
    @abstractmethod
    def make_sound(self):
        pass

class Dog(Animal):
    def make_sound(self):
        return "Woof!"

dog = Dog()
print(dog.make_sound())  # Woof!

68. What is a lambda function?

A lambda function is an anonymous function defined using the lambda keyword.

add = lambda x, y: x + y
print(add(3, 5))  # 8

69. What is a generator in Python?

A generator is a special type of iterator that yields values lazily using the yield keyword.

def count_up_to(n):
    count = 1
    while count <= n:
        yield count
        count += 1

gen = count_up_to(5)
print(list(gen))  # [1, 2, 3, 4, 5]

70. What is the difference between a list comprehension and a generator expression?

List comprehensions return a full list, while generator expressions return an iterator.

# List comprehension
lst = [x * x for x in range(5)]
print(lst)  # [0, 1, 4, 9, 16]

# Generator expression
gen = (x * x for x in range(5))
print(list(gen))  # [0, 1, 4, 9, 16]
