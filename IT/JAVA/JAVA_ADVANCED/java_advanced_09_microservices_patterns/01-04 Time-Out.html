Time-Out Pattern Explanation & Demo

What is the Time-Out Pattern?

The Time-Out Pattern prevents the system from waiting indefinitely for a slow service to respond, ensuring system resources are not blocked.

How It Works:

  • ‚ö° Typically used with **Circuit Breaker** or **Retry mechanisms**.

Example Implementation

function withTimeout(promise, timeout) {
  return new Promise((resolve, reject) => {
    const timer = setTimeout(() => reject("‚è≥ Time-out exceeded!"), timeout);
    promise.then(response => {
      clearTimeout(timer);
      resolve(response);
    }).catch(error => {
      clearTimeout(timer);
      reject(error);
    });
  });
}
