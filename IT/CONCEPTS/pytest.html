pytest Overview and Example

Overview:

  • Installation:

    You can install pytest using pip:

    pip install pytest
  • Key Features:
    • Easy test discovery: pytest automatically discovers and runs all test functions in the specified directories.
    • Clear and concise syntax for writing tests.
    • Powerful fixtures: Fixtures provide a way to set up and tear down resources needed for testing.
    • Advanced features for parameterized testing, test marking, and more.
    • Extensible and customizable.

Example:

Consider a simple module calculator.py with a function add that adds two numbers:



def add(x, y):
    return x + y;

Now, you want to write a test for this add function using pytest. Create a file named test_calculator.py with the following content:



from calculator import add

def test_add():
    result = add(3, 7)
    assert result == 10

    result = add(-1, 1)
    assert result == 0

    result = add(-5, -5)
    assert result == -10

In this example:

  • The test function is named test_add, and it uses assert statements to check the expected results of the add function for different inputs.
  • Save this file in the same directory as your calculator.py module or in a directory that pytest can discover.

To run the tests, open a terminal and navigate to the directory containing your test file and the module to be tested. Then, run the following command:

pytest

pytest will automatically discover and run your tests, providing a summary of the results.

This is a basic example, and pytest supports many additional features, including fixtures, parameterized testing, markers, and plugins. You can customize your testing workflow to suit the needs of your project.

Always refer to the official documentation for the most accurate and up-to-date information: pytest documentation.
