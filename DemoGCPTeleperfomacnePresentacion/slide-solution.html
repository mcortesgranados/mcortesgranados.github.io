Propuesta de solución Google Cloud + Workspace

Integrar resiliencia híbrida, analytics de datos y herramientas de colaboración en una pila coherente con IA para Mercado Libre.

Pila técnica propuesta

  • Infraestructura híbrida: Anthos para orquestar GKE, Cloud Run y cargas AWS existentes con seguridad consistente.
  • Autoscaling: Cloud Run + Managed Instance Groups con políticas predictivas para los picos estacionales.
  • Datos + IA: Pub/Sub → Dataflow → BigQuery + Vertex AI genera insights de fraude, marketing y logística en < 2 segundos.
  • Workspace + Chrome: Equipos seguros, colaboración ágil y flujo de trabajo integrado con Speakership en Google Meet.
  • Seguridad: BeyondCorp Enterprise y Secure Cloud Interconnect para proteger data sensibles y APIS.

IA adaptada

Vertex AI + Generative AI Studio para personalización, recomendadores, scoring de fraude y automatización de campañas con IA causal.

Experiencia humana

Workspace conecta desarrolladores y product teams para reducir tareas operativas e impulsar innovación.

Resultados esperados

Reducción de costos operativos, escalabilidad sin fricción y decisiones guiadas por datos que sustentan la ventaja competitiva.

Esta arquitectura se presenta como base para discutir escenarios de implementación y licenciamiento durante la reunión.
